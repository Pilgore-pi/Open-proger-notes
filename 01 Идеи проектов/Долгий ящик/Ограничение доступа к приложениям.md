Допустим, есть несколько приложений, к которым нужно ограничить доступ. Поместим их в скрытую папку. Создадим программу, которая позволяет открыть проводник в созданной нами скрытой папке с приложениями, чтобы их открывать на ограниченное время (1-2 часа). 
И здесь 2 варианта:

1) по завершении времени + 5-10 минут для закрытия приложения (доигровки), приложение закрывается.
2) после истечения времени выдвигается штраф "нельзя открывать приложения через эту программу в следующей сессии". После чего штраф снимается

Программа должна быть чувствительна только к указанным приложениям. Приложение просто позволяет открывать скрытую папку.
Штраф можно снять с помощью пароля.
Приложение ведёт лог сессий этих приложений, при этом, если указанные приложения были открыты не через эту программу, то в логе это помечается "Illigal"

Можно ослеживать открытие приложений через доступ к списку процессов Windows.

## Оператор **while**

```csharp
while (condition) {...}
```

## Оператор **do-while**

```csharp
do {...} while (condition);
```

Сначала выполняется условие, затем проверяется условие.

## Оператор **for**

**Как ведут себя переменные объявленные внутри блока кода (цикла, условия)**

```csharp
for(int i = 0; i < 9; i++) {...}
```

Инициализация переменной, условие и изменение переменной необязательны:

```csharp
// Бесконечный цикл
for( ; ; ){
    AcceptClients();
}
```

Выйти из любого цикла можно с помощью оператора **break**

## Оператор **foreach**

> Оператор `foreach` можно применять к любому объекту, являющемуся коллекцией (строки, массивы, списки и т. д.)

В многомерных массивах (например `int [,]`) **foreach** проходит по каждому элементу. Во вложенных массивах (например `int [][]`) **foreach** проходит только по верхнему слою

> Переменная, объявленная в цикле существует только в контексте этого цикла. При этом объявление переменной происходит один раз в любом случае.

```csharp
var fibNumbers = new List<int> { 0, 1, 1, 2, 3, 5, 8, 13 };

foreach (int element in fibNumbers) {
    Console.Write($"{element} ");
}

// Output: 0 1 1 2 3 5 8 13
```

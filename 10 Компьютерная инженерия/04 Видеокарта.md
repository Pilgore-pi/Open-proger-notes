
**Видеокарта (Graphics Processing Unit, GPU, Графический процессор)** — специализированное устройство, содержащее графический процессор с системой охлаждения, предназначенное для обработки и вывода графической информации на экран. Видеокарта подключается к материнской плате через интерфейс PCIe (PCI Express) или устаревший AGP и работает параллельно с центральным процессором

В отличие от CPU, ориентированного на высокую скорость последовательных вычислений, GPU специализирован на параллельной обработке большого количества независимых задач. Современные графические процессоры содержат тысячи ядер (2048 и более), позволяющих одновременно обрабатывать миллионы пикселей и вершин геометрии. GPU содержит большой объём видеопамяти (2-24 ГБ), однако не требует большого объёма кэш-памяти (128-256 КБ), так как данные обрабатываются потоковым способом

## Типы видеокарт

Всего есть 2 вида видеокарт: **интегрированная** и **дискретная**

> **Интегрированная графика (IGP, Integrated Graphics Processor)** — графический процессор, встроенный в центральный процессор или чипсет материнской платы. Использует часть оперативной памяти компьютера в качестве видеопамяти

Интегрированная графика применяется в:

- Дешёвых офисных компьютерах
- Ноутбуках и нетбуках
- Смартфонах и планшетах
- Серверах без требований к графике

Преимущества:

- низкая стоимость
- низкое энергопотребление
- отсутствие необходимости в дополнительном охлаждении

Недостатки:

- низкая производительность
- использование оперативной памяти (медленнее видеопамяти)
- отсутствие собственной видеопамяти

> **Дискретная видеокарта** — полнофункциональное подключаемое устройство с собственным графическим процессором, видеопамятью и системой охлаждения

Дискретная видеокарта применяется в:

- Игровых компьютерах
- Рабочих станциях для 3D-моделирования
- Серверах для обработки видео
- Компьютерах для машинного обучения (CUDA, ROCm)

Преимущества:

- высокая производительность
- собственная быстрая видеопамять
- независимое охлаждение

Недостатки:

- высокая стоимость
- высокое энергопотребление
- выделение тепла

## Производители графических процессоров

Основные производители GPU:

1. **NVIDIA** — лидер рынка дискретных видеокарт, архитектуры CUDA (GeForce, RTX, Tesla, Jetson)

2. **AMD** — второй крупный производитель дискретных видеокарт, архитектура RDNA (Radeon RX, Radeon Pro)

3. **Intel** — производитель интегрированной графики (Intel UHD, Iris Xe) с 2020 года, дискретные видеокарты Arc (с 2022 года)

4. **Qualcomm** — Adreno GPU для мобильных устройств

5. **ARM** — Mali GPU для мобильных устройств

6. **Apple** — Apple Silicon GPU (M1, M2, M3) для собственных устройств

## Архитектура видеокарты

Видеокарта состоит из следующих компонентов:

**Графический процессор (GPU)**

- Содержит тысячи простых вычислительных ядер (CUDA cores для NVIDIA, Stream Processors для AMD)
- Разделён на потоковые мультипроцессоры (SM для NVIDIA, CU для AMD)
- Каждый мультипроцессор содержит собственный кэш L1 и регистры

**Видеопамять (VRAM)**

- Быстрая память, подключённая непосредственно к GPU
- Типы: GDDR5, GDDR5X, GDDR6, GDDR6X, HBM2, HBM3
- Объём: 2-24 ГБ для потребительских видеокарт, до 80 ГБ для профессиональных

**Контроллер памяти**

- Управляет доступом GPU к видеопамяти
- Определяет ширину шины памяти (128, 192, 256, 384 бит)

**Система охлаждения**

- Радиатор с рёбрами охлаждения
- Вентиляторы (обычно 2-3 шт.)
- Тепловые трубки для отвода тепла от GPU

**Система питания**

- Преобразователь напряжения (VRM)
- Конденсаторы и дроссели
- Разъёмы питания (6-pin, 8-pin, 12-pin)

**Интерфейс подключения**

- PCIe x16 (PCI Express 3.0, 4.0, 5.0)
- Передача данных между материнской платой и видеокартой
- Пропускная способность: 16 ГБ/с (PCIe 3.0), 32 ГБ/с (PCIe 4.0), 64 ГБ/с (PCIe 5.0)

**Выходы видеосигнала**

- HDMI (High-Definition Multimedia Interface)
- DisplayPort (DP)
- DVI (Digital Visual Interface, устаревший)
- VGA (аналоговый, устаревший)

## Характеристики видеокарты

### Ядра и потоки GPU

> **CUDA Core (NVIDIA)** — базовая вычислительная единица GPU NVIDIA, способная выполнять одну операцию с плавающей запятой за такт

> **Stream Processor (AMD)** — аналог CUDA Core для видеокарт AMD

Количество ядер GPU значительно больше, чем у CPU (2048-16384 ядер против 4-64 ядер), однако каждое ядро GPU значительно проще и медленнее, чем ядро CPU. Это позволяет GPU эффективно обрабатывать параллельные задачи (рендеринг пикселей, обучение нейросетей).

### Специализированные блоки

**Tensor Cores (NVIDIA)** — специализированные блоки (тензорные ядра) для матричных операций, используются в машинном обучении и научных вычислениях. Каждое тензорное ядро может выполнять матричное умножение 4×4 за один такт

**RT Cores (NVIDIA)** — специализированные блоки для трассировки лучей (ray tracing) в реальном времени, ускоряют вычисление освещения и отражений в 3D-графике

**Ray Accelerators (AMD)** — аналог RT Cores для видеокарт AMD RDNA

### Видеопамять

> **Видеопамять (VRAM)** — высокоскоростная память, подключённая непосредственно к GPU, используется для хранения текстур, моделей, фреймбуфера и промежуточных результатов вычислений

**Типы видеопамяти (от старого к новому):**

| Тип | Год | Пропускная способность | Применение |
| :-- | :-- | :--------------------- | :--------- |
| GDDR5 | 2008 | ~224 ГБ/с | Видеокарты 2013-2016 |
| GDDR5X | 2015 | ~480 ГБ/с | Видеокарты 2016-2018 |
| GDDR6 | 2018 | ~360-480 ГБ/с | Видеокарты 2018-2022 |
| GDDR6X | 2020 | ~576-960 ГБ/с | Видеокарты 2020-2024 |
| HBM2 | 2015 | ~1 ТБ/с | Профессиональные видеокарты |
| HBM3 | 2022 | ~1.46 ТБ/с | Профессиональные видеокарты (Instinct MI300) |

**Объём видеопамяти:**
- 2-4 ГБ — бюджетные видеокарты, офисные задачи
- 6-8 ГБ — игровые видеокарты среднего уровня
- 10-12 ГБ — игровые видеокарты высокого уровня
- 16-24 ГБ — профессиональные видеокарты, машинное обучение
- 48-80 ГБ — серверные видеокарты для AI

### Разрядность шины памяти

> **Разрядность шины памяти** — количество бит информации, передаваемых между GPU и видеопамятью за один такт, определяет пропускную способность памяти

Пропускная способность рассчитывается по формуле:

```text
Пропускная_способность = (Разрядность_шины / 8) × Частота_памяти × 2
```

**Типичные значения разрядности:**

| Разрядность | Класс видеокарты | Примеры |
| :---------- | :--------------- | :------ |
| 64 бит | Бюджетная | GTX 1030, RX 6500 XT |
| 128 бит | Среднего уровня | GTX 1660, RX 6600 |
| 192 бит | Высокого уровня | RTX 3070 Ti, RX 6800 |
| 256 бит | Флагманская | RTX 3090, RTX 4090 |
| 384 бит | Профессиональная | RTX 6000 Ada |

### Тактовая частота GPU

> **Базовая частота (Base Clock)** — стандартная частота работы GPU при нормальной нагрузке

> **Boost частота (Boost Clock)** — максимальная частота, на которую GPU может автоматически разогнаться при высокой нагрузке и низкой температуре

Типичные значения:
- Base Clock: 1200-1500 МГц
- Boost Clock: 1800-2500 МГц

Частота видеопамяти обычно ниже, чем частота GPU (например, 10000 МГц для GDDR6, что эквивалентно 5000 МГц эффективной частоте).

### Филлрейт (пиксельная производительность)

> **Филлрейт** — скорость, с которой видеокарта может отрисовать пиксели на экран, измеряется в гигапикселях в секунду (GP/s)

Филлрейт рассчитывается по формуле:

```text
Филлрейт = (Количество_ROP × Частота_GPU) / 1000
```

Где **ROP (Render Output Unit)** — блок вывода результатов, отвечает за запись пикселей в фреймбуфер.

**Пример:** видеокарта с 64 ROP и частотой 2000 МГц имеет филлрейт 128 GP/s (64 × 2000 / 1000).

Существует два вида филлрейта:

1. **Пиксельный филлрейт** — скорость отрисовки пикселей без применения эффектов
2. **Текстурный филлрейт** — скорость выборки текселей из текстур (зависит от количества текстурных блоков TMU)

### Пропускная способность шины

> **Пропускная способность шины памяти** — максимальное количество данных, которое может быть передано между GPU и видеопамятью в единицу времени, измеряется в ГБ/с

Пропускная способность зависит от:
- Разрядности шины памяти (64, 128, 256 бит)
- Частоты видеопамяти
- Типа видеопамяти (GDDR5, GDDR6, HBM2)

**Примеры:**
- GTX 1060 (192-bit GDDR5): 192 ГБ/с
- RTX 3080 (320-bit GDDR6X): 760 ГБ/с
- RTX 4090 (384-bit GDDR6X): 1008 ГБ/с

### TDP видеокарты

> **TDP (Thermal Design Power)** — максимальное тепловыделение видеокарты, измеряется в ватах (Вт), определяет требования к системе охлаждения и блоку питания

TDP видеокарты зависит от:
- Количества ядер GPU
- Тактовой частоты
- Напряжения питания
- Техпроцесса

**Типичные значения TDP:**

| Класс видеокарты | TDP | Примеры |
| :--------------- | :-- | :------ |
| Бюджетная | 25-75 Вт | GTX 1030, RX 6500 XT |
| Среднего уровня | 100-150 Вт | GTX 1660, RTX 3060 |
| Высокого уровня | 200-320 Вт | RTX 3080, RTX 4080 |
| Флагманская | 320-450 Вт | RTX 3090 Ti, RTX 4090 |
| Профессиональная | 250-500 Вт | RTX 6000 Ada, A100 |

## Питание видеокарты

Видеокарта получает питание двумя способами:

1. **Через разъём PCIe** — до 75 Вт (стандарт PCIe 3.0)
2. **Через дополнительные разъёмы питания** — 6-pin (75 Вт), 8-pin (150 Вт), 12-pin (500 Вт)

**Типичные конфигурации питания:**

| Видеокарта | PCIe | 6-pin | 8-pin | 12-pin | Всего |
| :--------- | :--- | :---- | :---- | :----- | :---- |
| GTX 1660 | 75 Вт | — | — | — | 75 Вт |
| RTX 3060 | 75 Вт | — | 1× | — | 225 Вт |
| RTX 3080 | 75 Вт | — | 2× | — | 375 Вт |
| RTX 4090 | 75 Вт | — | — | 1× | 575 Вт |

При подборе блока питания необходимо учитывать мощность видеокарты и других компонентов. Рекомендуется выбирать БП с запасом 20-30% от общего потребления.

## Интерфейсы подключения видеокарты

### PCIe (PCI Express)

> **PCIe (PCI Express)** — последовательный интерфейс высокоскоростной передачи данных между компонентами компьютера

**Поколения PCIe:**

| Поколение | Год | Пропускная способность (x16) | Примеры |
| :-------- | :-- | :--------------------------- | :------ |
| PCIe 2.0 | 2007 | 8 ГБ/с | Старые видеокарты |
| PCIe 3.0 | 2010 | 16 ГБ/с | GTX 1080, RX 480 |
| PCIe 4.0 | 2017 | 32 ГБ/с | RTX 3080, RX 6800 |
| PCIe 5.0 | 2019 | 64 ГБ/с | RTX 4090, RX 7900 XTX |
| PCIe 6.0 | 2022 | 128 ГБ/с | Будущие видеокарты |

Видеокарта использует **x16 конфигурацию** (16 линий), обеспечивающую максимальную пропускную способность. На материнской плате может быть несколько слотов PCIe x16, но при использовании нескольких видеокарт пропускная способность распределяется между ними (например, 2 видеокарты работают на x8 каждая).

### Выходы видеосигнала

**HDMI (High-Definition Multimedia Interface)**
- Цифровой интерфейс для передачи видео и аудио
- Версии: HDMI 1.4 (4K@30Hz), HDMI 2.0 (4K@60Hz), HDMI 2.1 (8K@60Hz, 4K@120Hz)
- Разъём: 19-контактный

**DisplayPort (DP)**
- Цифровой интерфейс, разработанный VESA
- Версии: DP 1.2 (4K@60Hz), DP 1.3 (8K@60Hz), DP 1.4 (8K@120Hz)
- Преимущества: выше пропускная способность, поддержка большего разрешения
- Разъём: 20-контактный

**USB-C (Thunderbolt 3/4)**
- Универсальный интерфейс, используется в некоторых профессиональных видеокартах
- Поддерживает видео, данные и питание в одном кабеле

**DVI (Digital Visual Interface, устаревший)**
- Цифровой интерфейс, используется в старых видеокартах
- Максимальное разрешение: 1920×1200@60Hz

**VGA (Video Graphics Array, устаревший)**
- Аналоговый интерфейс, используется в очень старых видеокартах
- Максимальное разрешение: 1920×1440@60Hz

## Технологии масштабирования и синхронизации

### Масштабирование разрешения

**DLSS (Deep Learning Super Sampling, NVIDIA)**
- Использует искусственный интеллект для увеличения разрешения
- Рендерит изображение в низком разрешении, затем увеличивает с помощью нейросети
- Поддерживается на видеокартах с Tensor Cores (RTX серия)
- Версии: DLSS 1.0, DLSS 2.0, DLSS 3.0 (с Frame Generation)

**FSR (FidelityFX Super Resolution, AMD)**
- Открытый стандарт масштабирования, работает на видеокартах разных производителей
- Использует математические алгоритмы вместо нейросетей
- Поддерживается на видеокартах RDNA и старше

**XeSS (Intel Xe Super Sampling)**
- Аналог DLSS для видеокарт Intel Arc
- Использует искусственный интеллект для масштабирования

### Синхронизация частоты кадров

**V-Sync (Vertical Sync)**
- Синхронизирует частоту кадров GPU с частотой обновления монитора
- Предотвращает разрывы изображения (tearing)
- Недостаток: может вызвать задержку (input lag)

**G-Sync (NVIDIA)**
- Технология адаптивной синхронизации для мониторов NVIDIA
- Монитор подстраивает частоту обновления под частоту кадров GPU
- Требует монитор с поддержкой G-Sync и видеокарту NVIDIA

**FreeSync (AMD)**
- Открытый стандарт адаптивной синхронизации
- Работает на мониторах с поддержкой FreeSync и видеокартах AMD
- Также поддерживается некоторыми видеокартами NVIDIA (начиная с RTX серии)

## Многовидеокартные конфигурации

### SLI (Scalable Link Interface, NVIDIA)

> **SLI** — технология NVIDIA, позволяющая использовать две или более видеокарты одновременно для увеличения производительности

**Требования:**
- Две или более видеокарты NVIDIA одного поколения (обычно одинаковые модели)
- Материнская плата с поддержкой SLI (два и более слота PCIe x16)
- Специальный кабель SLI для соединения видеокарт

**Режимы работы:**
- **AFR (Alternate Frame Rendering)** — каждая видеокарта рендерит отдельные кадры
- **SFR (Split Frame Rendering)** — каждая видеокарта рендерит часть экрана
- **Compatibility Mode** — видеокарты работают независимо

**Недостатки:**
- Поддержка SLI прекращена в новых видеокартах NVIDIA (последняя поддержка в RTX 20 серии)
- Прирост производительности 60-80% при использовании двух видеокарт (не 100% из-за накладных расходов)
- Требует поддержки в игре или приложении

### CrossFire (AMD)

> **CrossFire** — технология AMD, аналог SLI для видеокарт AMD

**Требования:**
- Две или более видеокарты AMD (не обязательно одинаковые)
- Материнская плата с поддержкой CrossFire
- Кабель CrossFire для соединения видеокарт (в новых видеокартах может не требоваться)

**Режимы работы:**
- **AFR (Alternate Frame Rendering)** — аналог SLI AFR
- **CFX (CrossFireX)** — улучшенный режим с лучшей балансировкой нагрузки

**Статус:** CrossFire также снята с поддержки в новых видеокартах AMD RDNA

### NVLink (NVIDIA)

> **NVLink** — высокоскоростной интерфейс NVIDIA для соединения видеокарт, используется в профессиональных и серверных видеокартах

**Характеристики:**
- Пропускная способность: 100 ГБ/с (NVLink 1.0), 200 ГБ/с (NVLink 2.0), 400 ГБ/с (NVLink 3.0)
- Используется в видеокартах Tesla, A100, H100 для машинного обучения и научных вычислений
- Позволяет видеокартам обмениваться данными напрямую без участия CPU

## Техпроцесс видеокарты

> **Техпроцесс** — размер транзистора в нанометрах (нм), определяет плотность размещения транзисторов на кристалле GPU

Чем меньше техпроцесс, тем:
- Больше транзисторов можно разместить на одном кристалле
- Меньше энергопотребление при одинаковой производительности
- Выше тактовая частота при одинаковом энергопотреблении
- Меньше тепловыделение

**Эволюция техпроцесса:**

## Основные характеристики видеокарты

При выборе видеокарты необходимо учитывать следующие параметры:

| Характеристика | Описание |
| :------------- | :------- |
| **Архитектура** | Поколение GPU (Ada, RDNA 3, Arc Alchemist и т.д.) |
| **Ядра GPU** | Количество CUDA Cores (NVIDIA) или Stream Processors (AMD) |
| **Видеопамять** | Объём (ГБ) и тип (GDDR6, GDDR6X, HBM2) |
| **Разрядность шины** | 128, 192, 256, 384 бит — влияет на пропускную способность |
| **Базовая / Boost частота** | Стандартная и максимальная частота GPU (МГц) |
| **Частота памяти** | Частота видеопамяти (МГц) |
| **Пропускная способность** | Скорость передачи данных между GPU и памятью (ГБ/с) |
| **TDP** | Тепловыделение (Вт) — определяет требования к охлаждению |
| **Требования к питанию** | Необходимые разъёмы питания (6-pin, 8-pin, 12-pin) |
| **Интерфейс** | PCIe 3.0/4.0/5.0 x16 |
| **Выходы видеосигнала** | HDMI, DisplayPort, USB-C |
| **Габариты** | Длина, высота, количество слотов (влияет на совместимость с корпусом) |
| **Система охлаждения** | Тип (воздушное, жидкое), количество вентиляторов |
| **Специальные блоки** | Tensor Cores (NVIDIA), RT Cores (NVIDIA), Ray Accelerators (AMD) |

## Интересные факты

- **NVIDIA H100** — профессиональная видеокарта для AI, содержит 80 ГБ видеопамяти HBM2e, 16896 CUDA Cores, пропускная способность памяти 3.35 ТБ/с, стоимость $40,000

- Видеокарта RTX 4090 содержит 16,384 CUDA Cores и 24 ГБ видеопамяти GDDR6X, потребляет до 575 Вт

- Первая видеокарта с поддержкой 3D-графики — **3dfx Voodoo** (1996) с 4 МБ видеопамяти, стоила $300

- Современные видеокарты используют более 50 миллиардов транзисторов на кристалле размером 1-2 см²

- GPU содержит в 10-100 раз больше ядер, чем CPU, но каждое ядро медленнее в 10-100 раз, что позволяет эффективно обрабатывать параллельные задачи

- **CUDA** (Compute Unified Device Architecture) — платформа параллельных вычислений NVIDIA, позволяет использовать видеокарты для научных вычислений и машинного обучения

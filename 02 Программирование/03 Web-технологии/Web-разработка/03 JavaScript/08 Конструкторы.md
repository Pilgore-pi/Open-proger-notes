Можно создавать объекты с одинаковой структурой с помощью **функции-конструктора**:

```js
function User(name){
	this.name = name,
	this.isAdmin = false
}

let user = new User("Jack");
```

Существует 2 соглашения:

1. Имя конструктора должно начинаться с заглавной буквы
2. Функция должна выполняться только с помощью оператора `new`

```js
function User(name) {
  // this = {};  (неявно)

  // добавляет свойства к this
  this.name = name;
  this.isAdmin = false;

  // return this;  (неявно)
}
```

> Так как лямбды не могут использовать `this`, они не могут быть конструкторами.

Безымянный конструктор:

```js
// создаём функцию и сразу же вызываем её с помощью new
let user = new function() {
  this.name = "John";
  this.isAdmin = false;
};
```

Обычно конструкторы не имеют оператора `return`. Их задача – записать все необходимое в `this`, и это автоматически становится результатом.

Но если `return` всё же есть, то применяется простое правило:

- При вызове `return` с объектом, вместо `this` вернётся объект.
- При вызове `return` с примитивным значением, оно проигнорируется.

>Если у конструктора нет параметров, скобки необязательны. `new User;`

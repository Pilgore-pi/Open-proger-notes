Это структура данных, основанная на классе Array, которая расширяет возможности массива. Доступ к элементу происходит за O(1) (в общем случае в «связном списке» доступ осуществляется за O(N)).

Список в C# представляет обобщенный класс `List<T>`

```cs
List<int> ints = new List<int>(){ 1, 2, 3 };
ints.Add(4);
ints.Add(5);
```

Данные хранятся в скрытом массиве, который автоматически расширяется вдвое при переполнении. Начальный размер этого массива можно задать в одном из конструкторов (List(int capacity)).

```cs
// Конструктор копирования
List<int> Numbers = new List<int>(ints);

Numbers.Remove(4);   // удаление элемента по значению
Numbers.RemoveAt(3); // удаление элемента по индексу

//List: 1 2 3
```

При удалении элементы скрытого массива смещаются для заполнения пустой ячейки, а последний освободившийся элемент принимает значение default.

| Свойство   | Описание                         |
| :--------- | :------------------------------- |
| `Capacity` | текущая ёмкость скрытого массива |
| `Count`    | количество добавленных элементов |

| Метод                            | Описание                                  |
| :------------------------------- | :---------------------------------------- |
| `Add(T item)`                    | Добавляет элемент                         |
| `AddRange(IEnumerable<T> items)` | Добавляет список элементов в коллекцию    |
| `Reverse()`                      | Разворачивает весь список                 |
| `Contains(T item)`               | Проверяет на содержание элемента в списке |
| `IndexOf(T item)`                | Возвращает индекс заданного элемента      |
| `Insert(T item)`                 | Вставка элемента по индексу               |
| `Sort()`                         | Сортировка элементов в списке             |
| `Clear()`                        | Очищение списка                           |
| `Remove(T item)`                 | удаление элемента                         |
| `RemoveAt(int index)`            | Удаляет элемент по индексу                |

## Двухсвязный список. Класс `LinkedList<T>`

Класс `LinkedList<T>` представляет собой совокупность узлов (`LinkedListNode<T>`), каждый из которых содержит ссылки на предыдущий и на следующий узлы. Каждый узел обертывается в класс `LinkedListNode<T>`

Свойства класса `LinkedListNode<T>`:

- `Value` – значение узла
- `Next` – ссылка на следующий узел
- `Previous` – ссылка на предыдущий элемент

Свойства `LinkedList<T>`:

- `Count` – количество элементов
- `First` – первый элемент в списке
- `Last` – последний элемент в списке

Методы `LinkedList<T>`:

- **Вставка элемента:** `AddAfter()`, `AddBefore()`, `AddFirst()`, `AddLast()`
- **Удаление элемента:** `Remove()`, `RemoveFirst()`, `RemoveLast()`

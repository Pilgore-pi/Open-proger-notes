Вот подробный и качественный конспект по условным выражениям в языке Python, составленный по аналогии с твоим примером по C# (с сохранением форматирования и стиля повествования):

> В Python условные операторы `if-elif-else` являются **операторами**, а тернарный условный оператор (или условное выражение) является **выражением**, возвращающим значение.

Переменную типа `bool` можно инициализировать такими выражениями:

```python
is_true = (5 > 6) or (4 < 6)
```

## Конструкция `if-elif-else`

```python
# Полный синтаксис с блоками кода
if a > b:
    print("a > b")
elif a == b:
    print("a == b")
else:
    print("b > a")

# Оператор без фигурных скобок, отступы определяют блоки
if a > b:
    print("a > b")
else:
    print("b >= a")

# Можно писать однострочные if с использованием двоеточия и отступа
if a > b: print("a > b")
else: print("b >= a")
```

Переменные, объявленные внутри блоков `if`, `elif` или `else`, доступны во всей области видимости функции или модуля, так как Python не имеет локальной области видимости в блоках условных операторов (в отличие от C#).

## Конструкция аналогичная `switch-case`

В Python нет встроенного оператора `switch-case`, но похожее поведение можно реализовать через цепочку `if-elif-else`:

```python
if choice == 1:
    print(a + b)
elif choice == 2:
    print(a - b)
elif choice == 3:
    print(a * b)
else:
    print(0)
```

Или с помощью словаря и функции:

```python
def switch_case(choice):
    return {
        1: a + b,
        2: a - b,
        3: a * b,
    }.get(choice, 0)

choice = int(input())
print(switch_case(choice))
```

Для прерывания циклов используется оператор `break`, для перехода к следующей итерации — `continue`

## Тернарное условное выражение

> Тернарное условное выражение — это **выражение**, которое возвращает одно из двух значений в зависимости от условия (синтаксис: `[значение_если_истина] if [условие] else [значение_если_ложь]`)

```python
x, y = 3, 2
operation = input()

z = (x + y) if operation == "+" else (x - y)

# Размещение частей на новых строках допустимо для читаемости
z = (x + y) if operation == "+" \
    else (x - y)
```

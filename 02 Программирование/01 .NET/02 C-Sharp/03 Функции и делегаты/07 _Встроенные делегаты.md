В пространстве имен `System` определено несколько заготовленных делегатов, которые освобождают от необходимости создания собственного делегатного типа

Виды встроенные делегаты:

- **`Action`** — представляет любую процедуру, действие
- **`Func`** — представляет любую функцию
- **`Predicate`** — представляет предикат, логическую функцию
- **`EventHandler`** — делегат-шаблон, используемый в [[08 События|событиях]]

* `void Action()` — процедура без параметров
* `void Action<in T>(T t)` — процедура с параметром
* `TResult Func<out TResult>()` — функция
* `TResult Func<in T, out TResult>(T t)` — функция с одним параметром
* `bool Predicate<in T>(T t)` — предикат с одним параметром

В **`Func`** последний универсальный параметр — возвращаемое значение, остальные универсальные параметры — параметры функции.

Максимальное число входных параметров для **`Action`** & **`Func`** — 16. Для **`Predicate`** может быть только 1 входной параметр.

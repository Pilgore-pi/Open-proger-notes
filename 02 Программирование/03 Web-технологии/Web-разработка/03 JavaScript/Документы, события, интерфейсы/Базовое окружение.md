DOM (Document Object Model) — объектная модель документа, которая представляет все содержимое страницы в виде объектов, которые можно менять.

Например:
```js
// заменим цвет фона на красный,
document.body.style.background = "red";

// а через секунду вернём как было
setTimeout(() => document.body.style.background = "", 1000);
```

BOM (Browser Object Model) – это дополнительные объекты, предоставляемые браузером (окружением), чтобы работать со всем, кроме документа.

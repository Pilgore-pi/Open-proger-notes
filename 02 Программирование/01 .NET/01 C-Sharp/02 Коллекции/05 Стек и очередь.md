
> Класс `Queue<T>` представляет собой структуру данных “очередь”, работающую по принципу FIFO (First In First Out). Для очереди, как структуры данных, разрешены операции добавления в конец и удаления из начала.

При создании объекта Queue можно указать начальную емкость или массив, как для класса `List<T>`.
Количество элементов содержит свойство `Count`.

| Метод | Описание |
| :-- | :-- |
| Enqueue(T item) | добавление элемента |
| Dequeue() | удаляет элемент и возвращает его |
| TryDequeue(out T result) | попытка удалить элемент |
| Peek() | возвращает первый элемент |
| TryPeek(out T result) | попытка вернуть первый элемент |
| Contains(T item) |  |
| Clear() |  |

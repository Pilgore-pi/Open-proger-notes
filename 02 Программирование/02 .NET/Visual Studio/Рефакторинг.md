
## Вынесение класса в отдельный файл

В пункте "Быстрые действия и рафакторинг" можно выбрать "Вынести класс "MyClass" в отдельный класс MyClass.cs". 

## Создание конструктора на основе свойств

При создании класса можно выделить нужное количество свойств и создать конструктор, который будет инициализировать их. Это делается в пункте "Быстрые действия и рафакторинг".
```cs
internal class MyClass
{
	public MyClass(int myNumber, string myStr, DateTime myDate)
	{
		MyNumber = myNumber;
		MyStr = myStr;
		MyDate = myDate;
	}
	
	public int MyNumber { get; set; }
	public string MyStr { get; set; }
	public DateTime MyDate { get; set; }
}
```

## Добавление пространств имен

Если вы пишите тип, который находится в другом пространстве имет, можно нажать `Shift + Enter + Enter`, тогда необходимое пространство имен автоматически подключится.

## Создание необъявленной сущности кода

Если вы вводите название необъявленной сущности кода, у которой можно однозначно можно вывести тип, то следует нажать `Alt + Enter`, чтобы создать пустой метод, свойство и т. д. (ТРЕБУЕТСЯ ПРОВЕРКА)

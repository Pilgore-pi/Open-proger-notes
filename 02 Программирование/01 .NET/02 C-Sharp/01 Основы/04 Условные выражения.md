
▣ Конструкция if-else

◉ Переменная, объявленная внутри блока if-else существует только внутри области видимости этого блока. После выхода из области видимости переменная становится недоступной.

## ▣ Конструкция switch-case

|   |
|---|
|static int DoOperation(int op, int a, int b)<br><br>{<br><br>     switch (op)<br><br>     {<br><br>           case 1: return a + b;<br><br>           case 2: return a - b;<br><br>           case 3: return a * b;<br><br>           default: return 0;<br><br>     }<br><br>}|

  
Классический способ:

▽ **op** называется выражением соответствия. Выражение соответствия может иметь любой тип, но при этом не может быть **null**.

Блок **case** называется шаблоном. Существует 2 вида шаблонов:

1)    case constant:

2)    case type varname:

2) – компилятор проверяет, можно ли преобразовать выражение соответствия в данный тип **type** и выполняется, если преобразование возможно*.

|   |
|---|
|static int DoOperation(int op, int a, int b)<br><br>{<br><br>     return op switch<br><br>     {<br><br>           1 => a + b,<br><br>           2 => a - b,<br><br>           3 => a * b,<br><br>           _ => 0<br><br>     };<br><br>}|

  
Сокращенный способ:

◉ Также можно поместить результат работы блока switch-case в переменную.

|   |
|---|
|int DoOperation(int op, int a, int b) => op switch<br><br>{<br><br>     1 => a + b,<br><br>     2 => a - b,<br><br>     3 => a * b,<br><br>     _ => 0<br><br>};|


## Браузерные диалоговые окна

### Окно с оповещением

> Окно `Alert` представляет собой модальное диалоговое окно браузера для вывода информации пользователю

Alert блокирует выполнение скрипта до тех пор, пока пользователь не нажмет кнопку "OK"

Простой alert:

```javascript
alert("Привет, мир!");
```

Alert с переменной:

```javascript
let message = "Операция завершена";
alert(message);
```

Alert с нестроковыми значениями:

```javascript
alert(42);
alert(true);
alert({name: "John"});
```

### Ввод из диалогового окна браузера

> Браузер предоставляет встроенные функции для взаимодействия с пользователем через диалоговые окна

Простое окно ввода строки:

```javascript
let name = prompt("Введите ваше имя:");
console.log("Ваше имя: " + name);
```

Окно ввода со значением по умолчанию:

```javascript
let age = prompt("Введите ваш возраст:", "18");
console.log("Ваш возраст: " + age);
```

Подтверждение для получения логического значения:

```javascript
let result = confirm("Вы согласны?");
if (result) {
    console.log("Пользователь согласился");
} else {
    console.log("Пользователь отказался");
}
```

## Вывод в консоль

> Консоль является основным инструментом для вывода информации и отладки в JavaScript

Вывод в консоль осуществляется через объект `console`, который предоставляет различные методы логирования

### Основные методы консоли

Вывод обычного сообщения:

```javascript
console.log("Сообщение");
console.log(42);
console.log(true);
console.log({name: "John", age: 30}); // объект с двумя свойствами: имя (name) и возраст (age)
```

Вывод различных видов сообщений:

```javascript
console.info("Информационное сообщение");

console.warn("Это предупреждение");

console.error("Это ошибка");
```

Вывод таблицы данных:

```javascript
console.table([
    {name: "Alice", age: 25},
    {name: "Bob", age: 30}
]);
```

Вывод с форматированием:

```javascript
console.log("Имя: %s, Возраст: %d", "John", 30);
```

### Группировка логов

Создание группы логов:

```javascript
console.group("Группа 1");
console.log("Сообщение 1");
console.log("Сообщение 2");
console.groupEnd();
```

Создание свернутой группы:

```javascript
console.groupCollapsed("Свернутая группа");
console.log("Скрытое сообщение");
console.groupEnd();
```

### Ввод из консоли (Node.js)

объект console не предоставляет инструментов для ввода значений через консоль из-за ограничений функционала. Существует специальная программная среда Node.js, которая позволяет делать ввод через консоль

> В окружении Node.js ввод данных осуществляется через встроенный модуль `fs`

```javascript
const fs = require('fs');
const input = fs.readFileSync(0, 'utf-8');
console.log("Вы ввели:", input);
```

## Работа с файлами (Node.js)

> Работа с файлами в Node.js осуществляется через встроенный модуль `fs`

### Чтение файла

```javascript
const fs = require('fs');
const data = fs.readFileSync('file.txt', 'utf-8');
console.log(data);
```

### Запись в файл

```javascript
const fs = require('fs');
fs.writeFileSync('file.txt', 'Содержимое файла');
```

### Добавление в файл

Добавление содержимого в конец файла:

```javascript
const fs = require('fs');
fs.appendFileSync('file.txt', '\nНовая строка');
```

### Удаление файла

```javascript
const fs = require('fs');
fs.unlinkSync('file.txt');
```

```csharp
typeof(T).GetProperties(BindingFlags.Public | BindingFlags.Instance);
```

[Official](https://learn.microsoft.com/en-us/dotnet/api/system.reflection.bindingflags?view=net-8.0)

| Значение | Что означает | Пример использования |
| --- | --- | --- |
| `Default`| 1 | |
| `Public` | Включает публичные свойства или методы | Получение всех публичных свойств |
| `NonPublic` | Включает приватные, защищённые и внутренние свойства или методы | Получение приватных свойств |
| `Instance` | Включает нестатические свойства или методы | Получение свойств экземпляра объекта |
| `Static` | Включает статические свойства или методы | Получение статических методов класса |
| `DeclaredOnly` | Исключает свойства / методы унаследованные, ищет только объявленные в текущем типе | Только свойства, объявленные в данном классе |
| `FlattenHierarchy` | Включает унаследованные публичные статические свойства/методы | Для поиска статических свойств по наследованию |
| `IgnoreCase` | Игнорирует регистр имён при поиске | Поиск свойства с любым регистром имени |
| `OptionalParamBinding` | Используется в комбинированных сценариях с другими флагами, реже применяется | |

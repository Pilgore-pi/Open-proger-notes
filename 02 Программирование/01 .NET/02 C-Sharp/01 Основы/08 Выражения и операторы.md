
> Во всех современных языках программирования есть 2 основных типа сущностей кода:

- **Оператор** — это инструкция или команда программы, которая предписывает компьютеру выполнить определённое действие
- **Вычисляемое выражение** — последовательность инструкций программы (объявление переменных, арифметические операции, вызовы функций, использование операторов и т. д.), которая обязательно возвращает какой-то результат, который далее можно обрабатывать

## Примеры кода

```csharp
// Оператор присваивания: присваивает значение 5 переменной a
int a = 5;

// Оператор присваивания, справа — вычисляемое выражение (3 + 2 вычисляется в 5)
int b = 3 + 2;

// Оператор присваивания; справа — выражение умножения a и b
int c = a * b;

// Оператор вызова функции — выполняет действие вывода
Console.WriteLine(c);

// Выражение (сложение), вычисляется
a + b;
```

Более сложные конструкции:

```csharp
// Условный ОПЕРАТОР
// Здесь оператор в зависимости от условия выполняет
// первый или второй вызов функции
if (a > b) {
    Console.WriteLine("a > b");
}
else {
    Console.WriteLine("b >= a");
}

// Аналогичное условное ВЫРАЖЕНИЕ
// Здесь тернарный оператор возвращает строку, которую мы выводим
Console.WriteLine(a > b ? "a > b" : "b >= a");
```

Начиная с версии C\# 7.0 была введена функциональная версия оператора `switch-case`. Сам оператор `switch-case` — это просто разновидность условной конструкции, которая не возвращает ничего. Новая версия оператора `switch` теперь является *выражением* и всегда возвращает какой-то результат или генерирует исключение

```csharp
int choice = int.Parse(Console.ReadLine());

// традиционный опертор switch-case
switch (choice) {
    case 1: return a + b;
    case 2: return a - b;
    case 3: return a * b;
    default: return 0;
}

// оператор switch на основе паттерна Matching
return choice switch {
    1 => a + b,
    2 => a - b,
    3 => a * b,
    _ => 0
};
```

Наличие возможности использовать различные подходы к программированию в C\# демонстрирует проявление мультипарадигменности (наличия нескольких парадигм). В данном случае, смещение в сторону использо

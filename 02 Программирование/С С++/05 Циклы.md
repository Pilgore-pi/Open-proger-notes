
## Оператор **while**

```cpp
while (condition) {
    // тело цикла
}
```

Цикл выполняется, пока условие истинно. Условие проверяется перед каждой итерацией, если оно ложно — цикл завершается.

## Оператор **do-while**

```cpp
do {
    // тело цикла
} while (condition);
```

Сначала выполняется тело цикла, затем проверяется условие. Цикл гарантировано выполнится хотя бы один раз.

## Оператор **for**

```cpp
for (int i = 0; i < 9; i++) {
    // тело цикла
}
```

В блоке `for` можно указать:

- Инициализацию переменной
- Условие продолжения
- Изменение переменной (итерация)

Все части необязательны, например:

```cpp
// Бесконечный цикл for
for (;;) {
    AcceptClients();
}
```

## Управление циклом

- Выйти из цикла можно с помощью оператора **break**.
- Пропустить текущую итерацию и перейти к следующей — оператором **continue**

## О переменных цикла

Переменные, объявленные внутри заголовка цикла `for` видимы только внутри цикла

## Цикл for, основанный на диапазонах

В C++ можно использовать цикл `for` с диапазоном (range-based for):

```cpp
int fibNumbers[] = {0, 1, 1, 2, 3, 5, 8, 13};

for (int element : fibNumbers) {
    std::cout << element << " ";
}

// Output: 0 1 1 2 3 5 8 13
```

- В многомерных массивах циклы `for` или `range-for` надо вкладывать вложенно
- Переменная `element` существует только в теле цикла
- **В `C` не поддерживается**

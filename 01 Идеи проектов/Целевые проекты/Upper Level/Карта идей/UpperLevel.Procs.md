
### UpperLe

Подумать над ретранслятором C# в Kotlin. 

Используется анализатор кода C# для проверки корректности кода C#, чтобы он был совместим с Kotlin. Не весь код C# и Kotlin будет совместим рассматривается некоторое пересечение операторов, механизмов и библиотек обоих языков. Для библиотек потребуется создавать отдельные ретрансляторы

Использовать кастомный шаблон проекта "Kotlin retranslator", который использует какой-нибудь App builder, который может использовать консоль для логов, но выходным продуктом такого типа приложения будет текстовый файл или каталог со сгенерированными исходниками Kotlin

```csharp
class Human;
class Doctor : Human; //наслед.

Doctor ginecolog = new Doctor();
Human masha = ginecolog; 

//приведение
Doctor terapevt = (Doctor)masha; 
```

В маше, экземпляре базового класса, есть скрытые поля класса-наследника. Они раскрываются в экземпляре класса-наследника(terapevt) при его инициализации машей.

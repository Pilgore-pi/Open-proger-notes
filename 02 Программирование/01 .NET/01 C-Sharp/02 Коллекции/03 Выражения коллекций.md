С выходом C\# 12 появился новый синтаксис выражений коллекций. Позволяющий в кратком виде описывать коллекции, тип которых выводится компилятором по контексту.

```cs
// Массив
int[] arr = [1, 2, 3];

// Список
List<int> list = [1, 2, 3];

// Более сложный пример с вложением
int[][] matrix = [[1, 2], [3, 4]];

var matrix = [[1, 2], [3, 4]]; // тип невозможно вывести (ошибка компиляции)
```

`Nothing` — наследник всех вышестоящих типов, не имеет ни одного значения

Все числовые типы являются знаковыми и представляют типы значения.
Byte (1 байт), Short(2), Int(4), Long(8), Float(4), Double(8), BigInt($\infty$), BigDecimal($\infty$)

Числовые литералы:

```scala
10, -78, 0x54c, 2_100_000, 85l(85L), 12.3f, BigInt(25) 
```

Логические:

```scala
true, false
```

`Unit` — полноценный тип данных, имеющий единственное значение `()`
## Операции

`+, -, *, /, %`

Побитовые операции с числами: `|(ИЛИ), &(И), ~(НЕ)`

Логические: `||, &&, , ==, !=`

Все текстовые типы имеют кодировку UTF-16

Банально о строках... и `"""буквальное восприятие строки"""`

Интерполяция:

```scala
println(s"x = $x")
```

В Scala строки сравниваются содержимым:

```scala
"abc" == ("a" + "bc") // true
```

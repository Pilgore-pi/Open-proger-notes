
## Ввод-вывод в Python

### Ввод из консоли

Для получения данных от пользователя используется функция `input()`.

```python
name = input("Введите ваше имя: ")  # Запрос строки от пользователя
age = int(input("Введите ваш возраст: "))  # Ввод числа, преобразуем результат в int
```

- `input()` возвращает строку (тип `str`).
- Если нужно получить число, нужно явно преобразовать строку, например с помощью `int()`, `float()`.

### Вывод в консоль

Для вывода информации используется функция `print()`.

```python
print("Привет, мир!")  # Печатает строку в консоль
print(f"Имя: {name}, Возраст: {age}")  # Интерполяция строк (f-строки)
print("Сумма:", 5 + 3)  # Можно выводить несколько значений через запятую
```

- Функция `print()` автоматически добавляет перенос строки в конце.
- Можно менять разделитель и окончание с помощью параметров `sep` и `end`:

```python
print("A", "B", "C", sep="-")  # Выведет A-B-C
print("Без переноса", end="...")
print("продолжение")
```

### Файловый ввод-вывод

#### Открытие файла

Для работы с файлами используется функция `open()`, которая возвращает объект файла (file object). Ее синтаксис:

```python
file = open("имя_файла", "режим", encoding="кодировка")
```

- Режимы открытия:
  - `'r'` — чтение (по умолчанию)
  - `'w'` — запись (перезаписывает файл)
  - `'a'` — добавление в конец файла
  - `'x'` — создание файла, если существует — ошибка
  - `'b'` — бинарный режим (добавляется к другим режимам, например `'rb'`)
  - `'t'` — текстовый режим (по умолчанию)
  
- Параметр `encoding` обычно задают `"utf-8"` для текстовых файлов.

#### Чтение из файла

- Чтение всего файла целиком:

```python
with open("file.txt", "r", encoding="utf-8") as file:
    content = file.read()  # content — строка с содержимым файла
```

- Чтение по строкам:

```python
with open("file.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())  # .strip() удаляет пробельные символы и переносы
```

#### Запись в файл

- Записать строку (перезаписать файл):

```python
with open("file.txt", "w", encoding="utf-8") as file:
    file.write("Привет, файл!\n")
```

- Добавить в конец файла:

```python
with open("file.txt", "a", encoding="utf-8") as file:
    file.write("Дополнительная строка\n")
```

#### Особенности `with`

- Используется для автоматического закрытия файла после работы с ним.
- Исключает необходимость явно вызывать `file.close()`

### Пример: Ввод от пользователя и запись в файл

```python
name = input("Введите имя: ")
age = input("Введите возраст: ")

with open("output.txt", "w", encoding="utf-8") as file:
    file.write(f"Имя: {name}\n")
    file.write(f"Возраст: {age}\n")

print("Данные сохранены в файл.")
```

### Форматирование строк

В Python также есть возможности для форматирования строк:

| Метод            | Пример                                           | Описание                     |
|------------------|-------------------------------------------------|------------------------------|
| f-строки         | `f"Число {num}, текст {text}"`                  | Интерполяция с выражениями    |
| Метод `.format()` | `"Число {}, текст {}".format(num, text)`        | Форматирование с позициями   |
| `%` форматирование| `"Число %d, текст %s" % (num, text)`            | Старый стиль форматирования  |

Синтаксис класса:

```python
class название_класса:
    атрибуты_класса
    методы_класса
```

Пустой класс без атрибутов и методов:

```python
class Person:
    pass
```

Создание объекта (экземпляра) класса:

```python
tom = Person()      # определение объекта tom
bob = Person()      # определение объекта bob
```

## Конструкторы



```python
class Person:
    # конструктор
    def __init__(self):
        print("Создание объекта Person")
 
tom = Person()      # Создание объекта Person
```

## Атрибуты

> **Конструктор** — это специальный метод класса, который вызывается при создании объекта класса

```python
class Person:
 
    def __init__(self, name, age):
        self.name = name    # имя человека
        self.age = age        # возраст человека
 
 
tom = Person("Tom", 22)
 
# обращение к атрибутам
# получение значений
print(tom.name)     # Tom
print(tom.age)      # 22
# изменение значения
tom.age = 37
print(tom.age)      # 37
```

## Деструкторы

> **Деструктор** — это специальный метод класса, который вызывается при удалении объекта из памяти сборщиком мусора. Сборщик мусора удаляет объект, когда на него нет ссылок в программе, но, фактически, невозможно предугадать, в какой момент произойдет удаление в силу особенностей работы сборщика мусора.

Назначение деструктора — выполнение служебных операций при удалении объекта, например, очистка ресурсов. В целом, необходимость в использовании деструктора возникает очень редко, так как есть более удобные и управляемые механизмы очистки ресурсов, такие как контекстные менеджеры с методами `__enter__()` и`__exit__()`

```python
class Person:
  
    def __init__(self, name):
        self.name = name
        print("Создан человек с именем", self.name)
     
    def __del__(self):
        print("Удален человек с именем", self.name)
  
tom = Person("Tom")
```


## Базовые типы данных

### Числа

**Целые числа:**

| Тип данных | Диапазон       | Размер (байт) |
| ---------- | -------------- | ------------- |
| byte       | `[0, 255]`     | 1             |
| sbyte      | `[-128, 127]`  | 1             |
| short      | `[0, 255]`     | 2             |
| ushort     | `[0, 255]`     | 2             |
| int        | `[2^31, 2^31)` | 4             |
| uint       | `[0, 2^32)`    | 4             |
| long       | `[-2^, 255]`   | 8             |
| ulong      | `[0, 255]`     | 8             |

**Числа с плавающей точкой:**

| Тип данных | Описание | Размер (байт) |
| ---------- | -------- | ------------- |
| float      |          | 4             |
| double     |          | 8             |

**Числа с фиксированной точкой:**

| Тип данных | Описание | Размер (байт) |
| ---------- | -------- | ------------- |
| decimal    |          | 16            |

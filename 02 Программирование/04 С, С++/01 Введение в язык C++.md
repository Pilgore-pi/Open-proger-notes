> **Язык программирования C** — это процедурный язык программирования общего назначения

> **Язык программирования C++** — это объектно-ориентированный язык программирования общего назначения, разработанный Бьярном Страуструпом в 1983 году как расширение языка C.

Автор языка — Бьярн Страуструп (Bjarne Stroustrup), датский программист, который продолжает активно участвовать в развитии и стандартизации языка по сей день.

## Язык C

Язык C является:

* Компилируемым
* Императивным
* Низкоуровневым (с элементами высокоуровневого языка)
* Процедурным
* Статически типизированным (тип переменной не может меняться)
* Слабо типизированным (поддерживает неявное преобразование базовых типов)
* Явно типизированным (тип переменной указывается явно в коде)
* Языком общего назначения
* Минималистичным (небольшой набор ключевых слов и конструкций)

На [GitHub](https://github.com/torvalds/linux) исходный код ядра Linux, написанный на языке C, является одним из самых известных примеров использования этого языка

## Язык C++

Язык C++ является:

* Компилируемым
* Императивным
* Высокоуровневым (с возможностью низкоуровневого программирования)
* Мультипарадигменным:
    * Процедурным
    * Объектно-ориентированным
    * Функциональным
    * Обобщённым (Generic Programming)
    * Метапрограммированием
* Статически типизированным
* Слабо типизированным
* Явно типизированным
* Языком общего назначения
* Обратно совместимым с языком C (в большинстве случаев)

На [GitHub](https://github.com/gcc-mirror/gcc) находится исходный код компилятора GCC в открытом виде

## Различия между C и C++

| Характеристика         | C                                                       | C++                             |
| ---------------------- | ------------------------------------------------------- | ------------------------------- |
| Парадигма              | Процедурная                                             | Мультипарадигменная             |

| Пространства имён      | Нет                                                     | Да                              |
| Сложность              | Простой по набору конструкций, но сложный для понимания | Сложный во всех смыслах         |
| Производительность     | Высокая                                                 | Высокая (с небольшим оверхедом) |

## Компиляторы

Языки C и C++ не привязаны к какой-либо единой платформе, в отличие от C#, подчиняющегося .NET. Вместо этого они используют различные компиляторы и стандарты:

**Основные компиляторы:**

- GCC (GNU Compiler Collection) — кроссплатформенный компилятор
- Clang — компилятор, часть проекта LLVM
- MSVC (Microsoft Visual C++) — компилятор от Microsoft
- Intel C++ Compiler — компилятор от Intel

**Стандарты языков:**

- **C**: C89/C90, C99, C11, C17, C23
- **C++**: C++98, C++03, C++11, C++14, C++17, C++20, C++23, C++26

## История и стандартизация

C++ развивается через серию стандартов, каждый из которых добавляет новые возможности:

| Стандарт | Год | Основные особенности |
| -------- | --- | -------------------- |
| C++98    | 1998 | Первый стандарт ISO |
| C++03    | 2003 | Исправления и уточнения |
| C++11    | 2011 | Лямбда-функции, auto, умные указатели |
| C++14    | 2014 | Улучшения C++11 |
| C++17    | 2017 | Структурированные привязки, std::optional |
| C++20    | 2020 | Концепции, корутины, модули |
| C++23    | 2023 | Дальнейшие улучшения и оптимизации |

## Особенности C++

### Управление памятью

C++ предоставляет прямой контроль над управлением памятью:

- Ручное выделение и освобождение памяти (new/delete)
- Указатели и ссылки
- Умные указатели (smart pointers) в современном C++
- Стек и динамическая память

### Производительность

- Минимальные накладные расходы абстракций ("zero-cost abstractions")
- Возможность написания кода близкого к ассемблеру
- Отсутствие встроенной сборки мусора (контролируется программистом)
- Оптимизация на уровне компилятора

### Стандартная библиотека (STL)

C++ включает мощную Стандартную библиотеку шаблонов:

- Контейнеры (vector, list, map, set)
- Алгоритмы (sort, find, transform)
- Итераторы
- Функциональные объекты

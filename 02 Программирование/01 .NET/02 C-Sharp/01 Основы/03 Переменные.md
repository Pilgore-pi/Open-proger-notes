
## Базовые типы данных

### Числа

**Целые числа:**

| Тип данных | Диапазон        | Размер (байт) |
| ---------- | --------------- | ------------- |
| byte       | `[0, 255]`      | 1             |
| sbyte      | `[-128, 127]`   | 1             |
| short      | `[-2^15, 2^15)` | 2             |
| ushort     | `[0, 2^16)`     | 2             |
| int        | `[-2^31, 2^31)` | 4             |
| uint       | `[0, 2^32)`     | 4             |
| long       | `[-2^63, 2^63)` | 8             |
| ulong      | `[0, 2^64)`     | 8             |

**Числа с плавающей точкой:**

| Тип данных | Диапазон                                                 | Размер (байт) |
| ---------- | -------------------------------------------------------- | ------------- |
| float      | от $±1.5\ \times \ 10^{−45}$ до $±3.4\ \times\ 10^{38}$  | 4             |
| double     | от $±5.0\ \times\ 10^{−324}$ до $±1.7\ \times\ 10^{308}$ | 8             |

**Числа с фиксированной точкой:**

| Тип данных | Описание                                            | Размер (байт) |
| ---------- | --------------------------------------------------- | ------------- |
| decimal    | $±1.0\ \times\ 10^{−28}$ до $±7.9\ \times\ 10^{28}$ | 16            |

Системы контроля версий — это программное обеспечение для облегчения работы с изменяющейся информацией. Система управления версиями позволяет хранить несколько версий одного и того же документа, при необходимости возвращаться к более ранним версиям, определять, кто и когда сделал то или иное изменение, и многое другое.

В программировании СКМ используются для создания контрольных точек для кода программы, к которым можно вернуться при необходимости. И для совместной разработки приложения.

СКМ бывают ***локальными***, ***централизованными*** и ***распределенными***.
* Локальная использует одно устройство
* Централизированная использует один сервер
* Распределенные — общее облачное хранилище + локальные ПК разработчиков

>Репозиторий — это хранилище данных.

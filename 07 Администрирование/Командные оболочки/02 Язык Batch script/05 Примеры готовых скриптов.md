
Здесь приведены примеры скриптов, которые я сам использовал. Все скрипты протестированы на практике

## Перезапуск программы

```batch
@echo off

set PROCESS_NAME=MyProgram.exe
set PROCESS_PATH="C:\Path\To\MyProgram.exe"

taskkill /f /im %PROCESS_NAME%

start "" %PROCESS_PATH%
```

## Ярлык для запуска 1С:Предприятие

Запуск конфигуратора 1С с указанием логина и пароля:

```batch
@echo off
chcp 65001
"C:\Program Files\1cv8\common\1cestart.exe" CONFIG /F "C:\Path\To\InfoBase" /N "Admin" /P "123"
```

Запуск пользовательского приложения 1С с указанием логина и пароля:

```batch
@echo off
chcp 65001
"C:\Program Files\1cv8\common\1cestart.exe" ENTERPRISE /F "C:\Path\To\InfoBase" /N "Admin" /P "123"
```

## Установщик службы

Скрипт копирует содержимое каталога из сетевого диска на локальный диск, устанавливает и запускает службу:

```
@echo off
chcp 65001

setlocal

:: Указываем исходный и целевой каталоги
set source_folder=\\MyNetwork\Disk\MyPrograms\MyProgramFolder\
set destination_folder=C:\
set service_path=C:\MyProgramFolder\ProgramClient.exe

:: Проверяем существование исходного каталога
if not exist "%source_folder%" (
    echo Исходный каталог "%source_folder%" не найден.
    pause
    exit /b 1
)

:: Проверяем существование целевого каталога и создаём его, если нужно
if not exist "%destination_folder%" (
    mkdir "%destination_folder%"
)

:: Копируем содержимое каталога
xcopy "%source_folder%" "%destination_folder%" /e /h /k /q /r /c /y

echo Копирование файлов завершено.

start "%service_path%" /install
sc.exe start ProgramClient

pause
```

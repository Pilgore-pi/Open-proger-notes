
Запустим контейнер готового образа с DockerHub. Для этого нужно выполнить следюущие шаги:

1. Скачать приложение Docker Desktop с официального сайта

2. Запустить приложение

3. В строке поиска нужно ввести название тестового образа "`welcome-to-docker`"

4. Найденный образ можно скачать ("`Pull`") или скачать и запустить ("`Run`")

5. При запуске образа отобразится окно подтверждения, где можно будет задать настройки для контейнера, такие как название и используемый IP порт. В качестве порта для контейнера `welcome-to-docker` стоит указать `8088`, что указано в оффициальной документации

6. Проверка работоспособности: в строке браузера нужно ввести `localhost:8088`, после чего должна открыться страница с поздравлением и анимациями

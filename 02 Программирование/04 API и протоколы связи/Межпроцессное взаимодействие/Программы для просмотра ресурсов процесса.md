# Какие существуют программы для просмотра ресурсов процесса и его переменных? Я хочу сделать мод для однопользовательской простой игры, но у меня нет доступа к исходному коду игры

  
Для просмотра ресурсов процесса и его переменных, особенно если у тебя нет доступа к исходному коду игры, существуют специальные программы и отладчики, которые позволяют анализировать и мониторить процессы на уровне системы и памяти.

  

## Основные типы программ для мониторинга процессов и их ресурсов

  

### 1. Отладчики (Debuggers)

  

Отладчики позволяют изучать исполняемые файлы, просматривать переменные, регистры процессора, стек вызовов и изменять поведение программы во время выполнения.

  

- **x64dbg** — мощный отладчик для 32- и 64-битных приложений Windows с открытым исходным кодом. Он позволяет исследовать исполняемые файлы без исходного кода, изучать память процесса, регистры, точки останова и многое другое. Очень полезен для реверс-инжиниринга и создания модов к играм без доступа к исходникам[^1_1].

  
  

### 2. Мониторы процессов и системные анализаторы

  

Эти программы показывают информацию о запущенных процессах, их использовании ресурсов, открытых файлах, реестре и сетевой активности.

  

- **Process Monitor (Procmon)** — расширенный монитор процессов от Microsoft Sysinternals, который показывает в реальном времени действия с файловой системой, реестром и потоками процесса. Позволяет фильтровать события, просматривать подробные свойства операций и анализировать поведение процесса на глубоком уровне. Очень полезен для изучения того, как игра работает с ресурсами и данными[^1_2].

- **Process Explorer** — тоже из набора Sysinternals, дает детальную информацию о процессах, включая открытые дескрипторы, загруженные библиотеки, использование ресурсов и цифровые подписи. Можно использовать для общего анализа процессов и поиска связанных с игрой ресурсов[^1_3].

- **Process Hacker** — бесплатная и открытая альтернатива с расширенными возможностями мониторинга процессов, сетевой активности, управления службами и драйверами, а также инструментами для отладки[^1_4].

  
  

### 3. Программы для мониторинга аппаратных ресурсов (температура, FPS и т.п.)

  

Хотя они не показывают переменные процесса, могут быть полезны для оценки производительности игры.

  

- **Fraps**, **HWMonitor**, **HWiNFO**, **AIDA64** — для мониторинга FPS, температуры и других параметров, но не для просмотра переменных процесса[^1_5].

  
  

## Рекомендации для создания мода без исходного кода игры

  

- Используй **x64dbg** для отладки и изучения памяти процесса игры, чтобы найти нужные переменные и изменить их значения во время выполнения.

- Применяй **Process Monitor** для отслеживания операций с файлами и реестром, чтобы понять, какие данные игра загружает и сохраняет.

- Для анализа загруженных библиотек и ресурсов — **Process Explorer** или **Process Hacker**.

  

Эти инструменты в совокупности помогут тебе исследовать внутреннее состояние игры, найти интересующие переменные и ресурсы, а также внедрить изменения без доступа к исходному коду.

  

Если хочешь, могу помочь с конкретными инструкциями по работе с одним из этих инструментов.

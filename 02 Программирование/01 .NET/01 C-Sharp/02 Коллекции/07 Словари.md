> **Словарь**, представляемый классом `Dictionary<K, V>` — это неупорядоченная коллекция объектов, представляющих собой пару ключ-значение. Параметр `K` представляет тип ключа, `V` — тип значения. Каждый элемент словаря представляет собой структуру `KeyValuePair<K, V>`

Объявление словаря:

```cs
Dictionary<int, char> symbols = new();

var people = new Dictionary<int, string> {
    { 5, "Tom"},
    { 3, "Sam"},
    { 11, "Bob"}
};

var people = new Dictionary<int, string> {
    [5] = "Tom",
    [6] = "Sam",
    [7] = "Bob"
};
```

```cs
var mike = new KeyValuePair<int, string>(56, "Mike");

var employees = new List<KeyValuePair<int, string>>() { mike };

var people = new Dictionary<int, string>(employees);
//или

var people = new Dictionary<int, string>(employees) {
    [5] = "Tom",
    [6] = "Sam",
    [10] = "Bob",
};
//{"Mike", "Tom", "Sam", "Bob"}
```

> Обращение к элементам словаря осуществляется по ключу с помощью оператора индексирования

```cs
string sam = people[6];
```

| Метод                             | Описание                              |
| :-------------------------------- | :------------------------------------ |
| `Add(K key, V value)`             | добавление элемента                   |
| `TryAdd(K key, V value)`          | попытка добавить элемент              |
| `Remove(K key)`                   | удаление по ключу                     |
| `Remove(K key, out V value)`      | попытка удаления элемента             |
| `TryGetValue(K key, out V value)` | попытка получить значение по ключу    |
| `ContainsKey(K key)`              | проверка наличия элемента по ключу    |
| `ContainsValue(V value)`          | проверка наличия элемента по значению |
| `Clear()`                         | очищение словаря                      |

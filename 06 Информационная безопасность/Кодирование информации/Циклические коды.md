
Это разновидность линейных (блочных) кодов. Полученное кодовое слово при сдвиге
на N разрядов является правильным (любое N).

Преимущества:
1) Эффективность
2) Удобство реализации

Одним из наиболее известных и широко используемых циклических кодов является
код Боуза-Чоудхури-Хоквингема (BCH). Он является обобщением кодов Хэмминга и
обеспечивает более высокую степень исправления ошибок.

Другие циклические коды:

* БЧХ-коды (обобщение кодов Хэмминга)
* Коды Рида-Соломона (RS)
* Коды Фишера (двойное циклическое кодирование)
* Коды Терстона (модуляция, фазовая модуляция)
* Коды Хэмминга

Блочные коды — это кодирование сообщения, при котором данные подразделяются на блоки,
обрабатываемые независимо. Примером может послужить код Хэмминга.

Линейные коды — подкласс блочных кодов, которые используются для исправления
ошибок при передаче данных. К исходному сообщению добавляются проверочные биты,
которые вычисляются по линейным комбинациям битов сообщения, что делает их невосприимчивым
к ошибкам.

Восстановление данных с помощью линейных кодов происходит с высокой степенью надежности.
Код Хэмминга также является линейным.

# БЧХ-коды

это класс циклических кодов. Существуют параметры этого типа кодирования:
1) k — число информационных битов
2) t — максимальное число исправляемых ошибок. От этого параметра зависит сложность
   кодирования и декодирования

Для построения кода используется порождающий полином

# Сверточные коды

Это коды, отличающиеся от блочных кодов, тем, что работают с последовательностью, а не с
блоками. Основная идея сверточных кодов заключается в использовании конечного автомата,
который обрабатывает входные данные и генерирует кодовую последовательность на основе
текущего состояния автомата и входных символов. Это позволяет сверточным кодам иметь память
о предыдущих символах и контексте передачи данных.

Параметры сверточного кодирования:
1) скорость кодирования (количество кодовых символов, генерируемых в одном такте)
2) длина ограничивающего регистра (количество состояний в автомате)
3) Положение и количество обратных связей

Для эффективного декодирования используются алгоритмы Viterbi & BCJR.

Примечания:
* Hamming weight — in binary code the amount of the ones
* Hamming distance — количество несовпадающих битов между 2 кодами одинаковой длины
  в одинаковых позициях

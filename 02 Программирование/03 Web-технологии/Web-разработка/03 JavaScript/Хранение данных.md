
* Оперативная память
	1. JS — переменные обнуляются при перезагрузке страницы
	2. SessionStorage — выдержит перезагрузку страницы
* Память браузера
	1. Cookies — требуют домен, то есть не получиться запустить локально на компьютере. Имеют неприятный синтаксис
	2. LocalStorage (переменные хранят данные представленные в виде текста)
	3. Indexed data base
* Удаленный сервер (удаленно через AJAX) — Небезопасный способ. Аналоги:
	1. Adobe Flash (мертв)
	2. Silverlight (аналог Flash от Microsoft, мертв)
	3. Java-аплеты
	4. Web сокеты

Работа с cookie происходит через `document.cookie = "variable=value"`. При этом не происходит перезаписывания переменной cookie, а добавляется переменная со значением.

```js
document.cookie = 'user=John; expires=' + 'Sat, 01 Jan 2050 00:00:00: GTM'
```

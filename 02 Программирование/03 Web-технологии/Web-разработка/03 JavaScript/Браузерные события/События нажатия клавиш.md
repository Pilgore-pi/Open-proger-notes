[Источник](https://developer.mozilla.org/en-US/docs/Web/API/KeyboardEvent/code)

`document.onkeydown` -- событие нажатия клавиши, обработчик этого события может в качестве параметра иметь переменную `KeyboardEvent`.

Свойства `KeyboardEvent`:
* altKey: boolean
* ctrlKey: boolean
* code: string -- соответствует коду физической клавиши на клавиатуре (в раскладке QWERTY независимо от фактической раскладки). Примеры кодов: `Space`, `Backspace`, `KeyQ`, `KeyZ`, `ControlLeft`.
* key: string -- возвращает символ клавиши, который зависит от раскладки

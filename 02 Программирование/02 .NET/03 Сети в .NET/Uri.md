> **Uniform Resource Identifier (URI)** — символьная строка, которая глобально и уникально идентифицирует ресурс в сети

Uri является либо URL, либо URN, либо и тем и тем.

>URL — это URI, адрес ресурса в интернете, который помимо идентификации ресурса, предоставляет ещё и информацию о местонахождении этого ресурса

>URN — это URI, который только идентифицирует ресурс в определенном пространстве имен. Под пространством имен подразумевается набор имен и терминов, связанных тем или иным образом.

### Класс System.Uri

Представляет Uri строку

```csharp
Uri uri1 = new ("https://metanit.com", UriKind.Absolute);
Uri uri2 = new ("sharp/net", UriKind.Relative);
Uri uri3 = new ("https://metanit.com/sharp/net", UriKind.RelativeOrAbsolute);
```

UriKind — перечисление разновидностей Uri (все они представлены в примере)

Свойства:

* AbsoluteUri — абсолютный адрес Uri
* Authority — возвращает либо имя хоста в соответствии с системой доменных имен ==DNS==\*, либо IP-адрес и порт сервера
* Fragment — фрагмент адреса
* Host — хост
* Port
* Query — строка запроса текущего Uri
* Scheme — схема Uri
* IsFile — является ли Uri адресом файла
* IsLoopBack — указывает ли Uri на локальный хост
* OriginalString — возвращает строку, помещенную в конструктор перед инициализацией объекта
* Segments — массив частей пути Uri, разделенного слешами
* UserInfo — логин + пароль

Методы:

* TryCreate() — попытка создать Uri

Также существует класс-построитель Uri — UriBuilder\*.

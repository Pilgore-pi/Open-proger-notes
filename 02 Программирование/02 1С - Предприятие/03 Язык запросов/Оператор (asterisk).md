# Оператор `*`

Оператор `*` может использоваться в разных контекстах кода языка запросов

При выборе всех полей из источника данных

```bsl
ВЫБРАТЬ * ИЗ ИсточникДанных

select * from DataSource
```

При выборе всех полей указанного объекта

```bsl
ВЫБРАТЬ Справочник.Клиенты.*

select Catalog.Clients.*
```

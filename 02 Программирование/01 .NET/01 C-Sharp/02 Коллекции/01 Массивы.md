> **Массив** — это структура данных фиксированной длины, предназначенная для хранения набора элементов одного типа в непрерывной области памяти. Каждый элемент можно получить по индексу, отсчёт начинается с нуля.

- Все массивы являются ссылочными типами

Массивы бывают нескольких видов:

- Одномерные
- Многомерные
- Массивы массивов

Объявление одномерных массивов:

```cs
int[] numbers = new int[5]; // массив из 5 элементов типа int
numbers[0] = 0;
numbers[1] = 20;

string[] names = { "Anna", "Bob", "Tom" };

var names = new string[] { "Anna", "Bob", "Tom" };
```

Прямоугольный, двухмерный массив:

```cs
int[,] matrix = new int[3, 2] { {1, 2}, {3, 4}, {5, 6} };
matrix[1, 0] == 3; // доступ к элементам по индексам через запятую

var logicalMap = new bool[2, 4] {
    { true, true, false, false },
    { true, true, true , true }
};
```

Более многомерные массивы:

```cs
double[,,] cube = new double[2, 2, 2];
double[,,,] rotationMap = new double[10, 10, 10, 10];
```

Массивы массивов (еще их называют зубчатыми массивами (jagged array)):

```cs
int[][] jagged = new int[3][];
jagged[0] = new int[2];
jagged[1] = new int[3];
jagged[2] = new int[1];
jagged[1][0] = 7;      // доступ к элементам по вложенным индексам
```

Перебор массивов:

```cs
foreach (string name in names)
    Console.WriteLine(name);

for (int i = 0; i < names.Length; i++)
    Console.WriteLine(name);
```

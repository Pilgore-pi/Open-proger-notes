## Коллекции

### Массивы

> Все коллекции реализуют интерфейс `Iterable`

```scala
val res = Array(5, -2, 0)

println(res(1)) //-2
res(1) = -20
println (res(1)) //-20

val d = new Array[Double](8)
for x <- d do {
	println(x) //8 нулей (0.0)
}

// С нотацией типа:
a1: Array[Int] = Array(1, 2, 3)
```

> Обращение по индексу, на самом деле, — это вызов метода **apply**

```scala
// длина массива
a.length
// длина любой коллекции
a.size
// повернутая последовательность
a.reverse
// клонирует массив
a.clone
// проверка на содержание элемента
a.contains
// считает элементы с заданным условием
a.count(PREDICATE): Int
// возвращает неизменяемый массив с присоединенным массивом
a.appendedAll(a2)
// копирует в другой массив
a.copyToArray(a2)
// выбор элементов и выполнение преобразования (select LINQ)
a.map(x => x*3)
// выбор из уникальных элементов (.Select C#)
a.distinctBy(FUNC)
// возвращает новый массив в соответствии с функцией
a.map(FUNC)
// фильтрует согласно условию
a.filter(PREDICATE)

a.foreach(FUNC|ACTION)
//перебирает элементы и возвращает коллекции на основе этих
//элементов, склеенных в одну коллекцию
a.flatMap(T => Iterable)

a.min
a.max
a.sum
a.sorted
```

Преобразования:

```scala
a.toList
a.toMap
a.toArray
a.toSet
```

### Связные списки

> Элементы списка неизменяемы

```scala
val fruit1: List[String] = List("apples", "oranges", "pears")

val fruit2 = List("apples", "oranges", "pears")

val dim: List[List[Int]] =
   List(
      List(1, 0, 0),
      List(0, 1, 0),
      List(0, 0, 1)
   )
```

```scala
val myList = List(1,2,3)
println(0 :: myList) // List(0, 1, 2, 3) // prepend
```

> **`Nil`** — пустой список, наследник `List`

```scala
b.head    // первый элемент
b.tail    // все элементы, кроме первого
b.isEmpty

List.fill(size)(element) // создает size элементов = element
b.foreach(println) // принимает какую-то функцию и применяет ее к каждому элементу
var mult = 1;
val myList = List(10,20,30)

myList.foreach(x => mult *= x)
// ИЛИ сокращенно
myList.foreach(mult *= _)
```

### Set

`Set` — это множество. Элементы не повторяются. При попытке добавления в множество неуникального значения, это значение добавлено не будет и ошибки не возникнет

```scala
val set = Set(1,2,3)
val set2 = set + 4  //1 2 3 4
val set3 = set2 - 1 //2 3 4
```

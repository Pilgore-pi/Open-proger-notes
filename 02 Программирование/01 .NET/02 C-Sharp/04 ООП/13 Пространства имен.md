**Пространства имен (namespace) в C#** — это логические контейнеры, которые позволяют группировать связанные классы, интерфейсы, структуры, перечисления и другие типы, чтобы организовать код и избежать конфликтов имен. Они обеспечивают уникальность идентификаторов в пределах проекта и помогают структурировать большие проекты, делая код более читаемым и поддерживаемым

Объявление: используется ключевое слово namespace, за которым следует имя пространства имен. Внутри него размещаются классы и другие типы.

```csharp
namespace MyNamespace
{
    class MyClass
    {
        public void MyMethod()
        {
            Console.WriteLine("Пример пространства имен");
        }
    }
}
```

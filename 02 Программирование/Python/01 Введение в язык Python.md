**Язык программирования Python** — это высокоуровневый, интерпретируемый язык программирования общего назначения с динамической типизацией и поддержкой нескольких парадигм программирования. Его дизайн ориентирован на удобство чтения кода и краткость записи

Автор языка — Guido van Rossum, который начал разработку языка в конце 1980-х, а первая версия была выпущена в 1991 году

Язык Python является:

* Интерпретируемым
* Императивным
* Высокоуровневым
* Мультипарадигменным:
    * Объектно-ориентированным
    * Процедурным
    * Функциональным
* Динамически типизированным
* Сильно типизированным
* Явно типизированным (через динамическую проверку)
* Языком общего назначения
* Открытым и кроссплатформенным

Исходный код Python доступен на [GitHub](https://github.com/python/cpython) в составе проекта CPython — основной реализации языка

## Платформа и экосистема

Python не привязан к какой-то одной универсальной платформе, но обладает богатой стандартной библиотекой и широким сообществом, что делает его пригодным для самых разных платформ — Windows, macOS, Linux, а также встраиваемых систем, Raspberry Pi и др.

Для интеграции с другими платформами существуют разные реализации Python, например:

- `CPython` — основная и наиболее распространённая реализация
- `IronPython` — реализация на базе .NET
- `Jython` — для JVM (Java Virtual Machine)
- `PyPy` — с улучшенной производительностью благодаря JIT-компиляции

## Область применения

Python применяется в самых разных областях, включая:

- Веб-разработка (серверная часть, фреймворки Django, Flask)
- Научные вычисления и Data Science (библиотеки NumPy, Pandas, Matplotlib)
- Машинное обучение и искусственный интеллект (TensorFlow, scikit-learn, PyTorch)
- Автоматизация и администрирование
- Разработка настольных приложений (PyQt, Tkinter)
- Разработка игр
- Встраиваемые системы и IoT

Он удобен для быстрого прототипирования, а также для создания полноценных коммерческих продуктов

- Язык не содержит директив препроцессора

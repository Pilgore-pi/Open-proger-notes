
> В C\# условные операторы `if-else` и `switch-case` являются **операторами**, а не выражениями, а тернарный условный оператор и оператор сопоставления с образцом `switch` являются **выражениями**, возвращающими какое-то значение

Переменную типа `Boolean` можно инициализировать такими выражениями:

```csharp
bool IsTrue = (5 > 6) || (4 < 6);
```

```csharp
if (5 > 8) {
     Console.WriteLine("5 > 8!!!");
}

else Console.WriteLine("8 > 5");
```
  
▣ Конструкция if-else

◉ Переменная, объявленная внутри блока if-else существует только внутри области видимости этого блока. После выхода из области видимости переменная становится недоступной.

## Конструкция switch-case

```csharp
static int DoOperation(int op, int a, int b) {
    switch (op) {
        case 1: return a + b;
        case 2: return a - b;
        case 3: return a * b;
        default: return 0;
    }
}
```

Блоки `case` и `default` могут содержать блок кода неограниченного размера. Окончание блока `case` или блока `default` — это один из управляющих операторов:

- `break`
- `retur`

## Тернарная операция

> **Тернарная операция** (от слова "терра" - три (операнда)) — это **выражение**, которое возвращает одно из двух значений в зависимости от условия

```csharp
int x = 3, y = 2;

string operation = Console.ReadLine();

int z = operation == "+"? (x + y) : (x - y);

//condition? iftrue : iffalse
```

## Диапазоны

> **Диапазон** — это последовательность значений, определённая начальной и конечной точками. Диапазоны используются для итерации, проверки принадлежности значения и создания коллекций.

## Оператор диапазона

```kotlin
val range = 1..5
```

Оператор диапазона описывает последовательность, где крайние границы включены в последовательность

Цикл **`for-in`** с проходом по диапазону:

```kotlin
for (i in 1..5) {
    print("$i ")
}

// Output: 1 2 3 4 5
```

> Оператор `for-in` можно применять к любому объекту, являющемуся итерируемой коллекцией (строки, массивы, списки, диапазоны и т. д.)

> Переменная, объявленная в цикле существует только в контексте этого цикла. При этом объявление переменной происходит один раз в любом случае

```kotlin
val fibNumbers = listOf(0, 1, 1, 2, 3, 5, 8, 13)

for (element in fibNumbers) {
    print("$element ")
}

// Output: 0 1 1 2 3 5 8 13
```

Проверка условия "3 входит в диапазон 1..5"

```kotlin
if (3 in 1..5) {
    println("3 входит в диапазон")
}

// Output: 3 входит в диапазон
```

## Оператор **until (открытый диапазон)**

```kotlin
val range = 1 until 5
```

Включает начало, но исключает конец

```kotlin
for (i in 1 until 5) {
    print("$i ")
}

// Output: 1 2 3 4
```

> Оператор `until` полезен при работе с индексами массивов и списков, так как их размер часто используется как верхняя граница

```kotlin
val numbers = listOf(10, 20, 30, 40, 50)

for (i in 0 until numbers.size) {
    println("numbers[$i] = ${numbers[i]}")
}
```

## Оператор **downTo (обратный диапазон)**

```kotlin
val range = 5 downTo 1
```

Создаёт диапазон в обратном порядке (от большего к меньшему)

```kotlin
for (i in 5 downTo 1) {
    print("$i ")
}

// Output: 5 4 3 2 1
```

## Оператор **step (шаг диапазона)**

```kotlin
val range = 1..10 step 2
```

Определяет шаг итерации (по умолчанию шаг равен 1)

```kotlin
for (i in 1..10 step 2) {
    print("$i ")
}

// Output: 1 3 5 7 9
```

Работает с `downTo`:

```kotlin
for (i in 10 downTo 1 step 2) {
    print("$i ")
}

// Output: 10 8 6 4 2
```

Работает с `until`:

```kotlin
for (i in 0 until 10 step 3) {
    print("$i ")
}

// Output: 0 3 6 9
```

```kotlin

    println("m входит в диапазон")
}

// Output: m входит в диапазон
```

С шагом:

```kotlin

}

// Output: acegikmoqsuwy
```

## Функции для работы с диапазонами

## Оператор **repeat**

```kotlin
repeat(5) {
    println("Повторение $it")
}

// Output:
// Повторение 0
// Повторение 1
// Повторение 2
// Повторение 3
// Повторение 4
```

> Функция `repeat` выполняет блок кода указанное количество раз. Переменная `it` содержит текущий номер итерации (начиная с 0)


## Базовые типы данных

### Числа

**Целые числа:**

| Тип данных | Диапазон        | Размер (байт) |
| ---------- | --------------- | ------------- |
| byte       | `[0, 255]`      | 1             |
| sbyte      | `[-128, 127]`   | 1             |
| short      | `[-2^15, 2^15)` | 2             |
| ushort     | `[0, 2^16)`     | 2             |
| int        | `[-2^31, 2^31)` | 4             |
| uint       | `[0, 2^32)`     | 4             |
| long       | `[-2^63, 2^63)` | 8             |
| ulong      | `[0, 2^64)`     | 8             |

**Числа с плавающей точкой:**

| Тип данных | Диапазон                                                 | Размер (байт) | Точность     |
| ---------- | -------------------------------------------------------- | ------------- | ------------ |
| float      | от $±1.5\ \times \ 10^{−45}$ до $±3.4\ \times\ 10^{38}$  | 4             | 7 знаков     |
| double     | от $±5.0\ \times\ 10^{−324}$ до $±1.7\ \times\ 10^{308}$ | 8             | 15-16 знаков |

**Числа с фиксированной точкой:**

| Тип данных | Описание                                               | Размер (байт) | Точность     |
| ---------- | ------------------------------------------------------ | ------------- | ------------ |
| decimal    | от $±1.0\ \times\ 10^{−28}$ до $±7.9\ \times\ 10^{28}$ | 16            | 28-29 знаков |

| Тип данных | Описание                                                                                                                                                         |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| DateTime   | **Дата и время**; значение в диапазоне от 00:00:00 (полуночи), 1 января 0001 г. нашей эры (н. э.) до 11:59:59, 31 декабря 9999 г. в григорианском календаре      |
| DateOnly   | **Дата без времени**;  значение в диапазоне от 00:00:00 (полуночи), 1 января 0001 г. нашей эры (н. э.) до 11:59:59, 31 декабря 9999 г. в григорианском календаре |

## Именование переменных


> **Идентификатор переменной** — это уникальное имя переменной, по которому можно к ней обращаться

```csharp
int myVariable;
int my_variable;
int MY_VARIABLE;
int _my_variable;
int @my_variable; // Символ @ игнорируется
int \u005my_variable;
int @int; // Переменная с именем "int"
int myVariable@; ????
```

Символ **@** нужен для того, чтобы убрать неоднозначаность между оператором языка C\# и именем переменной. Этот символ позволяет называть переменные также, как и операторы языка. В коде всегда нужно писать символ **@**, если переменная объявлена с именем, содержащим этот символ

Недопустимые идентификаторы:

```csharp
int 111var;
int @@var;
int @6var;
int true;
int i∈Z;
```

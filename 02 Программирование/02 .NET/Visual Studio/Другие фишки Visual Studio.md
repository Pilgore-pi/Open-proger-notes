`Ctrl + -` – переводит экран на то место, где вы были ранее. Например, вы перешли в определение какого-то метода по нажатию `ctrl`, но затем хотите вернуться обратно, там где вы были, тогда следует использовать эти горячие клавиши. (минус, который рядом с нулем)

## Комментарии к сущностям кода
Существует возможность создавать комментарии к сущностям кода, которые будут видны при наведении на эту сущность. Это делается с помощью тега `<summary> ... </summary>`. Тег `<summary>` следует размещать перед сущностью:
```xml
/// <summary>
/// This method calculating amount of my finger IQ
/// </summary>
public void Method() {...}
```

Чтобы создать тег `<summary>`, нужно ввести `/// + Enter`.

Внутри тега `<summary>` можно объявить тег `<see cref="Reference"/>`. Этот тег делает текст, записанный в `cref` ссылкой на сущность кода.
```xml
/// <summary>
/// This is <see cref="MyClass"/> method that
/// calculates amount of my finger IQ
/// </summary>
public void Method() { }
```

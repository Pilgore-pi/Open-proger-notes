Управление консолью происходит через класс `Console`, предоставляющий необходимые методы для ввода-вывода в консоль

Самыми важными методами консоли являются лишь несколько:

- `Write()`
- `WriteLine()`
- `Read()`
- `ReadLine()`
- `ReadKey()`
- `Clear()`
- `ResetColor()`
- `ForegroundColor`
- `BackgroundColor`

| Метод                                  | Возвращает                | Описание                                                                                                                                                                  |
| -------------------------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Write()`                              | `void`                    | Выводит текстовое представление объекта в консоль                                                                                                                         |
| `WriteLine()`                          | `void`                    | Аналогичен `Write()`, но в конец, к выводу, добавляет перевод на новую строку                                                                                             |
| `Read()`                               | `int`                     | Считывает первый символ из введенной строки и возвращает код этого символа в кодировке UTF-16                                                                             |
| `ReadLine()`                           | `string?`                 | Считывает введенный пользователем текст как строку и возвращает ее                                                                                                        |
| `ReadKey()`                            | `(struct) ConsoleKeyInfo` | Считывает введенную клавишу и возвращает информацию о ней в виде структуры `ConsoleKeyInfo`. Параметр `bool intercept` позволяет не отображать вводимые клавиши в консоль |
| `Clear()`                              | `void`                    | Очищает буфер консоли                                                                                                                                                     |
| `Beep()`                               | `void`                    | Воспроизводит звуковой сигнал                                                                                                                                             |
| `ResetColor()`                         | `void`                    | Сбрасывает цвет текста и фона к стандартным значениям                                                                                                                     |
| `GetCursorPosition()`                  | `(int left, int top)`     | Возвращает позицию курсора как пару значений: колонка и строка. Минимальное значение для обеих позиций — 0                                                                |
| `SetCursorPosition(int left, int top)` | `void`                    | Задает позицию курсора. Позиция = колонка, строка. Минимальное значение позиции — 0                                                                                       |

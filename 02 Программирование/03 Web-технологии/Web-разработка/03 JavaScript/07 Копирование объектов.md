```js
let user = {
  name: "John",
  age: 30
};

let clone = {}; // новый пустой объект

// Копирование всех свойств в объект-клон
for (let key in user) {
  clone[key] = user[key];
}
```

`Object.assign(target: any, source: any)` Копирует все свойства источника или источников в целевой объект и возвращает целевой объект.

```js
let user = {
  name: "John",
  age: 30
};

let clone = Object.assign({}, user);
```

Упаковка в Object:

```js
let text = 'djf';
alert(typeof text); // string
text = Object.assign(text);
alert(typeof text); // object
```

>Если копируемый объект содержит свойство-объект, то его тоже нужно клонировать, то есть осуществить глубокое копирование. Это можно сделать рекурсивным перебором, но есть уже готовый метод глубокого копирования.

Библиотека `lodash` предоставляет методы копирования объектов.
* `_.clone()` копирует объект, состоящий из примитивных типов
* `_.cloneDeep()` копирует в том числе и вложенные объекты

Сравнение двух объектов по значениям
```js
_.isEqual(a, b);
```

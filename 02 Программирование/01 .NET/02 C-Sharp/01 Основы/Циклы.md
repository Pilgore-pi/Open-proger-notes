## Оператор **for**

**Как ведут себя переменные объявленные внутри блока кода (цикла, условия)**

```csharp
for(int i = 0; i < 9; i++) {...}
```

Инициализация переменной, условие и изменение переменной необязательны:

```csharp
// Бесконечный цикл
for( ; ; ){
    AcceptClients();
}
```

Выйти из любого цикла можно с помощью **break**.

▣ Оператор **foreach**

▽ Данный цикл применим к любому типу, реализующему интерфейс System.Collection.IEnumerable или System.Collection.Generic.IEnumerable<T>

В многомерных массивах **foreach** проходит по каждому элементу.

◉ Переменная, объявленная в цикле существует только в контексте этого цикла. При этом объявление переменной происходит один раз в любом случае.

▽ Существует возможность создавать блоки кода и располагать в них некоторый код. Такой блок будет обладать собственной областью видимости.

# Операторы перехода

### ■ Оператор **break**

Оператор прерывает исполнение ближайшего итерационного оператора (в котором оператор **break** находится) или оператора **switch****-****case**.

### ■ Оператор **continue**

Прерывает выполнение ближайшего итерационного оператора и начинает следующую итерацию.

### ■ Оператор **return**

Завершает выполнение метода и передает управление вызывавшему методу.

Оператор **return** можно вызывать в **void** методе.

### ■ Оператор **goto**

|   |
|---|
|for (int i = 0; i < 10; i++)<br><br>     for (int j = 0; j < 10; j++)<br><br>     {<br><br>           if (j == 5) goto Mark;<br><br>     }<br><br>Mark:<br><br>     Console.WriteLine("Transferred to mark");|

  
Передает управление оператору, помеченному меткой.

Данный оператор крайне не рекомендуется использовать, так как он приводит к запутыванию логики программы.

### ■ Оператор **throw**

Генерирует заданное исключение.

Синтаксис:

throw exception;

где exception – объект семейства System.Exception.

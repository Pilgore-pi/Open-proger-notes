## Описание функционала

Приложение со списком шуточных вопросов

Поддерживаемые операции: CRUD

- Create (Создание)
- Read (Чтение)
- Update (Редактирование)
- Delete (Удаление)

Шутки и ответы будут храниться в базе данных MS SQL Server

## Создание проекта по шаблону


В Visual Studio:
    `ASP.NET Core Web Application`
    Выберите версию .NET: `.NET Core`
    Выберите шаблон проекта 

В JetBrains Rider:
    `Web > Web App (Model-View-Controller)`
    Auth: `Individual authentication`

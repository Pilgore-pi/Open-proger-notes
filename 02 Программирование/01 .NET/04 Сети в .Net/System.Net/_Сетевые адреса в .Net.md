
В .Net IP адрес представлен классом System.Net.**IPAddress**.

Способы создания:
```csharp
// Конструкторы
public IPAddress(byte[] address);
public IPAddress(long address);
//long address от 0 до 0x0000_0000_FFFF_FFFF

var ip = IPAddress.Parse("127.0.0.1");
```

Порядок байт — Big endian (слева направо)

* Метод TryParse тоже существует

Встроенные адреса как статические свойства:

* Loopback (127.0.0.1)
* Any (0.0.0.0)
* Broadcast (255.255.255.255)

Свойство `enum AddressFamily` представляет семейство или схему адресации, к которому относиться данный IP

## IPEndPoint

Это комбинация IP-адреса и порта. Используется для связывания сокетов с локальными адресами или при соединении с удаленными адресами.

```csharp
public IPEndPoint (long address, int port);
public IPEndPoint (IPAddress address, int port);

IPEndPoint endPoint1 = IPEndPoint.Parse("127.0.0.1:8080");
IPEndPoint.TryParse(
	"127.0.0.1:8080", out IPEndPoint? endPoint2);
```

Свойства:
* Address — IP address
* Port — порт
* AddressFamily — схема адресации

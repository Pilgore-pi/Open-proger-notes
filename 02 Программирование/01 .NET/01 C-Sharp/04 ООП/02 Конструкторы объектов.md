> Каждый раз, когда создается класс или структура, вызывается конструктор

> **Конструктор** — это специальный метод класса, который неявно вызывается при создании объекта через ключевое слово `new`

> Конструктор не может иметь возвращающего значения как такового

## Стандартный конструктор

> Если программист не определил конструктор в классе или структуре, то компилятором ~~~~~C# (не средой .NET) будет определен **стандартный конструктор** без параметров неявным образом

## Специальный конструктор

> **Специальный конструктор** — это конструктор, определенный программистом

> **Конструктор копирования** — это конструктор, единственным параметром которого является ссылка на тип. В .NET такой вид конструкторов **не используется**, в отличие от таких языков, как C++

> Как только программист определяет специальный коструктор, стандартный конструктор удаляется из класса

Если класс/структура содержит только один приватный конструктор, то объект "снаружи" создать будет невозможно

## Деструкторы

Деструкторы, в подавляющем большинстве случаев, не используются, так как объекты удаляются автоматически сборщиком мусора

> Деструкторы вызываются перед тем, как объект будет удален из оперативной памяти

> Нужно избегать использование деструкторов если это возможно, так как сам факт использования деструктора где-либо в проекте значительно снижает производительность итоговой программы

## Примеры кода

В версии C# 9 появилась возможность вызова конструктора без указания типа, в случаях, когда компилятор имеет возможность вывести этот тип из контекста:

```csharp
Person tom = new Person();          // аналогично new Person();
var    bob = new Person("Bob");     // аналогично new Person("Bob");
var steven = new("Steven");         // Исключение на этапе компиляции, невозможно вывести тип

Person tom = new();          // аналогично new Person();
Person bob = new("Bob");     // аналогично new Person("Bob");
Person sam = new("Sam", 25); // аналогично new Person("Sam", 25);
```

В .NET все переменные явлются объектами, поэтому любую переменную можно создать через ключевое слово `new`

```csharp
int  i = new int(1);
char c = new char('p');
```

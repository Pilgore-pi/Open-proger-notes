>Сниппет -- шаблон, используемый с сочетаниями клавиш.

После применения сниппета, могуты быть созданы выделенные области по которым можно переключаться по `tab`, например имя итерирующей переменной в цикле `for`.

Любой **оператор** (if, while, else, unsafe, try...) + tab + tab, **тип** (class, interface, delegate...) + tab + tab и директива + tab + tab создают соответствующий шаблон, например:

```cs
// for
for (int i = 0; i < length; i++)
{

}

// forr
for (int i = length - 1; i >= 0 ; i--)
{

}

// enum
enum MyEnum
{

}
```

### propdp -- создание свойства зависимостей

```cs
public static int GetMyProperty(DependencyObject obj)
{
	return (int)obj.GetValue(MyPropertyProperty);
}

public static void SetMyProperty(DependencyObject obj, int value)
{
	obj.SetValue(MyPropertyProperty, value);
}

// Using a DependencyProperty as the backing store for MyProperty.
// This enables animation, styling, binding, etc...
public static readonly DependencyProperty MyPropertyProperty = DependencyProperty.RegisterAttached("MyProperty", typeof(int),
   typeof(ownerclass), new PropertyMetadata(0));
```

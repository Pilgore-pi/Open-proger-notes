## Платформа JVM

> Язык Kotlin подчиняется универсальной платформе разработки приложений Java Virtual Machine (JVM)

Платформа JVM определяет единую систему типов и берет на себя взаимодействие с ОС, предоставляя удобный программный интерфейс для разработки приложений. Kotlin полностью совместим с Java и может использовать все библиотеки экосистемы Java.

Существует несколько языков программирования, которые были разработаны на основе JVM:

- Kotlin
- Scala
- Groovy
- Clojure
- Java

Другие же языки напрямую не поддерживают стандарт JVM, но некоторые из них поддерживают интеграцию с JVM экосистемой:

| Язык   | Реализация с JVM |
| ------ | ---------------- |
| Python | Jython           |
| Ruby   | JRuby            |
| JavaScript | Nashorn (устарел), GraalVM |
| Lisp   | Clojure          |
| R      | FastR            |
| PHP    | Quercus          |

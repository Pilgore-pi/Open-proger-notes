## Области видимости

> **Область видимости** (scope) — это часть программы, в пределах которой доступна и видима определённая переменная или идентификатор. Вне этой области переменная перестаёт существовать или становится недоступной.

Существует несколько уровней областей видимости:

- Уровень сборки (проекта). В этой области видимости объявляются глобальные переменные на уровне разрабатываемой программы
- Уровень класса 

Переменная, объявленная внутри блока if-else существует только внутри области видимости этого блока. После выхода из области видимости переменная становится недоступной.

```csharp
{
    int jar = 308;
}

jar++; //не существует в текущем контексте
```
  
Существует возможность создавать блоки кода и располагать в них некоторый код. Такой блок будет обладать собственной областью видимости.

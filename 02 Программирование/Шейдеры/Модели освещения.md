# Ray casting

Луч проецируется из камеры. Когда луч пересекается с объектом, он окрашивается в цвет этого объекта. Каждый объект описывается функцией пересечений.

# Ray tracing

`Ray tracing` — это развитие модели `Ray casting`

Это Ray casting, но каждый луч теперь может отражаться от объектов.

# Ray marching

Объекты в сцене задаются аналитическим уравнением, описывающим кратчайшее расстояние между камерой и поверхностью этих объектов сцены. 

1) Берется ближайшая фигура и расстояние до нее (минимальное расстояние).
2) Выпускается множество лучей в каких-то направлениях на найденное расстояние. конец луча — новая точка. 

Из каждой новой точки выпускается луч и алгоритм повторяется, пока расстояние до объекта не будет достаточно малым (параметр точности). Тогда пиксель этой точки окрашивается в цвет объекта. Есть ещё параметр — количество итераций. Алгоритм может зацикливаться, поэтому число итераций должно быть ограничено.
Принято использовать знаковые функции расстояний — расстояние до границы объекта из точки, которая уже находится внутри этого объекта считается отрицательным. Если использовать без знаковые функции, то возможно появление артефактов.

Если на брать максимальное расстояние до объекта вместо минимального, то будут рендериться только пересечения объектов. Можно также менять знак функции, описывающей тело, тогда мы получим Дополнение объекта, все станет этим объектом, кроме области которую этот объект занимал раньше.
Сглаживание объектов: вместо min/max брать сглаженное значение min/max.

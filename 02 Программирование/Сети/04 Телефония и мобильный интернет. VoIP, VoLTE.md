## Поколения мобильного интернета

| Поколение | Дата выхода | Скорость передачи | Описание |
| :-- | :-- | :-- | :-- |
| 1G | начало 1980-х | До 2,4 Кбит/с | Первое поколение аналоговой мобильной связи, предоставлявшее базовые голосовые вызовы |
| 2G | начало 1990-х | От 14 до 217 Кбит/с | Переход на цифровую связь, появились SMS, улучшенное качество голосовой связи, GPRS |
| GPRS (`G`) |  | до 170 Кбит/с | Пакетная передача данных в сетях 2G |
| EDGE (`E`) |  | ~470 Кбит/с | Улучшенная версия GPRS, также работает в 2G и 2.5G |
| 3G | начало 2000-х | От 380 Кбит/с до 42 Мбит/с | Высокоскоростной мобильный интернет, поддержка видео-звонков и смартфонов |
| HSPA (`H`) |  | до ~5 Мбит/с | Технология 3G, обеспечивающая высокоскоростную пакетную передачу данных |
| HSPA+ (`H+`) |  | до ~50 Мбит/с | Расширение HSPA для увеличения скорости передачи данных, чуть медленнее, чем 4G |
| 4G (`LTE`) | около 2010 года | От 100 Мбит/с до 1 Гбит/с | Быстрый интернет, поддержка потокового видео, VoLTE – голосовые звонки через IP-сети |
| 4G+ (`LTE-A`) |  | до ~500 Мбит/с |  |
| 4.5G (LTE-Advanced Pro; Pre 5G) |  | до 1 Гбит/с | более продвинутая версия LTE-A, иногда называемая Pre-5G, которая поддерживает ещё более широкие возможности, включая агрегацию большего количества несущих, использование новых технологий модуляции и повышения пропускной способности |
| 5G | с 2020 года | От 2 Гбит/с до 20 Гбит/с | Очень высокая скорость и низкая задержка, поддержка IoT |
| 6G | ожидается к 2030 | ? | Гиперскоростные сети, ещё более низкая задержка |

Технологии XG (5G, 6G) и WiFi отличаются по нескольким ключевым аспектам:

| Аспект | Технологии XG (5G, 6G) | WiFi |
| :-- | :-- | :-- |
| Тип сети | Сотовые мобильные сети, покрытие больших территорий | Локальные беспроводные сети, ограниченное покрытие |
| Частотный спектр | Лицензируемый спектр (например, миллиметровые и терагерцовые волны) | Нелицензируемый спектр (2,4 ГГц, 5 ГГц и выше) |
| Управление и инфраструктура | Управляется операторами связи, требует базовых станций | Пользовательская инфраструктура (роутеры, точки доступа) |
| Мобильность | Обеспечивает полноценную мобильность и подключение в движении | Ограничена зоной покрытия точки доступа |
| Скорость и задержка | Очень высокая скорость (до терабит/с у 6G), низкая задержка | Высокая скорость локально, но может снижаться при загрузке |
| Цели использования | Мобильный интернет, IoT, VR/AR, автономные системы | Домашний и офисный интернет, локальный обмен данными |
| Оплата и лицензирование | Абонентская плата операторам, лицензии на частоты | Бесплатное использование нелицензируемых частот |

Таким образом, XG — это глобальные мобильные сети с высокой пропускной способностью и мобильностью, которые обеспечивают связь везде, в то время как WiFi — локальные сети высокой скорости, удобные для домашнего и офисного использования внутри ограниченной зоны.[^2][^3][^4][^8]

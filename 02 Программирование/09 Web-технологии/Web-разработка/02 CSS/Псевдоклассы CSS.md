Псевдоклассы это селекторы, после которых ставится двоеточие и указывается имя псевдокласса. Псевдоклассы позволяют более точно указывать стили для элементов.
```css
a:link { color: navy }
```

Псевдоклассы можно комбинировать с другими типами селекторов.

Псевдоклассы для ссылок:
* link — цвет "свежей" ссылки
* visited — цвет посещенной ссылки

Для чекбоксов:
* checked

Выбор всех дивов, кроме дива с id=unique_div:
```css
div:not(#unique_div) {
  color:blue;
}
```

* p::first-line — селектор первой линии текста
* p::first-letter — селектор первой буквы текста

## Событийные псевдоклассы
Позволяют определить, как будет выглядеть элемент при вызове события (наведение мыши, нажатие...)
```css
.message:hover{
	border: 2px solid purple
	color: white
}
```

Типы событий:
* hover — наведение курсором на элемент
* focus — получение фокуса элементом
* active — срабатывает, когда элемент активируется (нажатие кнопки)

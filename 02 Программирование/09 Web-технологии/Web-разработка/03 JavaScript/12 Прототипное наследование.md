```js
let animal = {
    eats: true
};
let rabbit = {
    jumps: true
};

rabbit.__proto__ = animal;
```

Свойство `__proto__` хранит прототип объекта. Если у `rabbit` отсутствуют искомые свойства, они будут браться из `animal`.

В цепи прототипного наследования не может быть циклов.

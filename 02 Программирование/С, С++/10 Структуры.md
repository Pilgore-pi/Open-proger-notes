
> Структура — это пользовательский, составной тип данных, который может содержать в себе произвольное количество компонентов с произвольными типами и именами

https://metanit.com/c/tutorial/6.1.php

Пример определения структуры:

```c
struct Person {
    char* name;
    int age;
};

struct BrainRot {
    int watchedVideos;
    int tikTokWatchPercentage;
};
```

Можно объявить структуру без определения, тогда структура будет являться неполным типом с неизвестным размером:

```c
struct Game;
```

## Инициализация структуры

```c
struct BrainRot {
    int watchedVideos;
    int tikTokWatchPercentage;
};

int main(void) {
    struct BrainRot brainRot = { 49557, 100 };

    printf("watched videos: %d \t TikTok watch percentage: %s"
        , brainRot.watchedVideos
        , brainRot.tikTokWatchPercentage
    );

    return 0;
}
```

Инициализация полей структуры отдельно:

```c
#include <stdio.h>
  
struct person {
    int age;
    char * name;
};
  
int main(void) {
    struct person tom;
    tom.name ="Tom";
    tom.age = 22;
    printf("Name:%s \t Age: %d\n", tom.name, tom.age);
    return 0;
}
```

Объявление глобальной переменной структуры:

```c
struct BrainRot {
    int watchedVideos;
    int tikTokWatchPercentage;
} rottenBrains;

// Определение сразу нескольких глобальных переменных
struct person {
    int age;
    char * name;
} tom, bob, alice;

int main(void) {
    rottenBrains.watchedVideos = 9000000;
    rottenBrains.tikTokWatchPercentage = 70%;
    
    return 0;
}
```

Объявление глобальной переменной структуры с инициализацией

```c
struct BrainRot {
    int watchedVideos;
    int tikTokWatchPercentage;
} rottenBrains = { 9000000, 70 };
```

Объявление через `typedef`

```c
#include <stdio.h>
 
typedef struct {
    int age;
    char* name;
} person;
 
int main(void) {
    person tom = {23, "Tom"}; // ключевое слово struct не требуется
    printf("Name:%s \t Age: %d\n", tom.name, tom.age);
    return 0;
}
```

Определение через макроподстановки:

```c
#include <stdio.h>
 
#define PERSON struct {int age; char name[20];}
 
int main(void) {
    PERSON tom = {23, "Tom"};
    printf("Name:%s \t Age: %d\n", tom.name, tom.age);
    return 0;
}
```

Структуры можно объявлять внутри функции, тогда структура не будет видна вне функции

```c
int main(void) {
    struct person {
        char * name;
        int age;
    };
    struct person tom = {"Tom", 22};
    printf("Name: %s  Age: %d\n", tom.name, tom.age);
    return 0;
}
```

В C++ поддерживаются модификаторы доступа к полям структуры. Они будут описаны далее при описании ООП в C++

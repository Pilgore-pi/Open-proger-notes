> **Словарь**, представляемый классом `Dictionary<K, V>` — это неупорядоченная коллекция объектов, представляющих собой пару ключ-значение. Параметр `K` представляет тип ключа, `V` — тип значения. Каждый элемент словаря представляет собой структуру `KeyValuePair<K, V>`

```cs
Dictionary<int, char> symbols = new();

var people = new Dictionary<int, string> {
	{ 5, "Tom"},
	{ 3, "Sam"},
	{ 11, "Bob"}
};
```

> **CSS** (Cascading Style Sheets, каскадные таблицы стилей) — это формальный язык описания внешнего вида документа, написанного с помощью языка разметки

В роле языка разметки, например, могут выступать: HTML, XHTML, XML

CSS код принято отделять в отдельный файл, хотя есть возможность использования кода CSS непосредственно в HTML коде

> Комментарии доступны только многострочные: `/*comment*/`

## Селекторы

**Селектор** — это тип тега, к которому применяется стиль. Стиль определяется с помощью свойств (нотация kebab-case):

```css
a {
	color: green;
	font-size: 16px;
}
```

`a` — это селектор HTML-тега "a". Указанный стиль будет применяться ко всем тегам на странице.

CSS файл необходимо связать с документом HTML с помощью тега `<link>`
При этом нужно указать значения следующих атрибутов:
- `href` — путь к файлу CSS (относительный или полный)
- `rel` — отношения с файлом (нужно установить в stylesheet)
- `type` — тип (нужно установить в text/css)

```html
<link rel="stylesheet" href="snake.css">
```

Количество файлов CSS для связывания со страницей не ограничено.

>Селектор `*` позволяет выбрать все элементы на странице.

#### Селекторы по идентификатору

Чтобы конкретно указать стиль для определенных элементов, для них определяется атрибут "`id`", который не должен начинаться с цифры. Далее в CSS идет обращение по id:

```css
#my_tag_a{
	color: green;
	font-size: 16px;
}
```

#### Селекторы по классу

У большинства тегов есть атрибут "`class`", который представляет собой название какого-то класса, классы могут повторяться для нескольких элементов. Таким образом можно применять CSS к тегам с указанным классом:

```css
.small{
	font-size: 14px
}

.big{
	font-size: 24px;
}
```

>У тегов может быть много классов, которые разделяются пробелом при указании атрибута "`class`", но определять несколько классов для элемента не рекомендуется.

>Если свойство для объекта страницы указано несколько раз, то будет учитываться последнее указанное значение.

### Селектор атрибута (используется редко)

Можно выбрать элемент по атрибуту или его части

```css
a[href="www.google.com"]{ color: #1f2333; }

/* По части атрибута: */
a[href*="google"]{ color: #AA00FF; }
```

### Вложенные селекторы

"`X Y`" позволяет выбрать всех потомков указанного элемента (тег, класс...)
Ниже выбираются все ссылки, находящиеся внутри списка:

```css
li a {
  text-decoration: none;
}
```

Селекторы `X > Y` аналогичны "`X Y`" селекторам, но выбираются только прямые потомки элемента.

## Комбинации селекторов

Во избежание дублирования кода можно объединять селекторы разных элементов через запятую, а затем, если нужно, указывать специфические характеристики для элементов

```css
h2, .block, .msg { background: #fff; }
```

#### Соседние теги (почти не используются)

```css
div~p{}
div+p{}
```

## Основные свойства блоков html

У любого блока HTML есть:
* border: 5px solid red — рамка
* padding(-left, -right, -top, -bottom) — внутренний отступ
* margin(-left, -right, -top, -bottom) — внешний отступ
* width — ширина
* height — высота

```css
width: 200px; /*необходимо указывать единицы измерения*/
width: 0; /*единственный случай, когда можно не указывать*/
width: 50%; /*проценты от родителя*/
padding-left: -100px;
```

#### Цвет

```css
color: #FF0000;
color: #F00; /*эквивалент #FF0000 (дублирование сиволов)*/
color: red; ПРОВЕРИТЬ
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5); /*alpha from 0 to 1*/
```

#### Картинка

```css

```

При указании картинки на фон, будет хорошим тоном указать также цвет, чтобы, если картинка еще не загрузилась отображался подходящий цвет.

#### Текст

```css
div{
	font-family: Arial, Tahoma, sans-serif;
	/*если невозможно подгрузить Arial, то Tahoma, если*/
	/*не Tahoma, то любой из семейства sans-serif*/
	font-size: 16px;
	font-style: italic;
	font-weight: bold;
	line-height: 1.5; /*полуторный интервал*/
	text-decoration: underline;
	text-align: center;
	vertical-align: middle;
}
```

## .NET Standard

**.NET Standard** — это официальная спецификация для API .NET, которые должны быть доступны во всех реализациях .NET.
.NET Standard была создана для того, чтобы повысить согласованность в экосистеме .NET.

В .NET 5 и более поздних версиях применяется другой подход к созданию единообразия, что устраняет необходимость в .NET Standard в большинстве сценариев

### 12.0

| Нововведение                                 | Описание                                                                          |
| -------------------------------------------- | --------------------------------------------------------------------------------- |
| Первичные конструкторы                       | Первичные конструкторы можно определять для любого класса или структуры           |
| Выражения коллекций                          | `[..collection1, ..collection2]`, `['a', 'b', 'c']`                               |
| Статические массивы (`InlineArray(SIZE)`)    |                                                                                   |
| Необязательные параметры в лямбда-выражениях |                                                                                   |
| `ref readonly` параметры                     |                                                                                   |
| Псевдонимы типов                             | `using Point = (int x, int y);`<br>`using Collection = System.Generics.List<int>` |
| Экспериментальные атрибуты                   | Предназначены для пометки сущности кода как экспериментальной функции             |

Реализация `Buffer`:

```csharp
[System.Runtime.CompilerServices.InlineArray(10)]
public struct Buffer
{
    private int _element0;
}

var buffer = new Buffer();
for (int i = 0; i < 10; i++)
{
    buffer[i] = i;
}

foreach (var i in buffer)
{
    Console.WriteLine(i);
}
```

### 13.0

| Нововведение                                                                                       | Описание                                                                                                      |
| -------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Расширение модификатора `params`                                                                   | Теперь можно применять модификатор к параметрам семейства `IEnumerable<T>`, а не только к массивам            |
| Новая `escape`-последовательность `\e`                                                             | `\e` = `ESCAPE` (`U+001B`, `\u001b`)                                                                          |
| Оператор индекса `^` теперь доступен в выражении инициализации объекта                             |                                                                                                               |
| Добавлено анти-ограничение для параметров обобщенных типов `allows ref struct`                     |                                                                                                               |
| Модификатор `partial` теперь доступен для свойств и индексаторов                                   |                                                                                                               |
| Контекстное ключевое слово `field`                                                                 | Позволяет не создавать явно поле для определяемого свойства, и обращаться к нему через ключевое слово `field` |
| `ref struct` типы могут реализовывать интерфейсы, но не могут быть упакованы в ссылку на интерфейс |                                                                                                               |

Оператор индекса в выражении инициализации объекта

```csharp
public class TimerRemaining
{
    public int[] buffer { get; set; } = new int[10];
}

var countdown = new TimerRemaining()
{
    buffer =
    {
        [^1] = 0,
        [^2] = 1,
        [^3] = 2,
        [^4] = 3,
        [^5] = 4,
        [^6] = 5,
        [^7] = 6,
        [^8] = 7,
        [^9] = 8,
        [^10] = 9
    }
};
```

## Правило 60-30-10

Это правило говорит о том, что:
- 60% пространства UI должны занимать базовые цвета;
- 30% вторичные цвета;
- 10% должен занимать яркий цвет

Базовые цвета нужны для фона;
вторичные для создания цветового разнообразия;
яркий цвет используется для интерактивных элементов или важных элементов.

### Realtime colors

Это [сайт для примерки различных дизайнов](https://realtimecolors.com/?colors=000000-ffffff-4685ff-f2f2f2-ffb084) по правилу 60-30-10. Нужно просто настроить понравившийся дизайн и экспортировать себе палитру, которая будет содержать цвета, разделенные по типам и выраженные в HEX и RGB.

## Цветовые модели

1) CYMK (Cyan, Magenta, Yellow, Black) — тип цветовой модели с вычитанием цветов (пересечение всех цветов = черный)
2) RGB (Red, Green, Blue) — тип со сложением цветов (сумма всех цветов = белый)
3) HSB (Hue, Saturation, Brightness)

>Теплые цвета обычно используются для выделения активных областей, а холодные цвета — для выделения пассивных областей. 

## Фишки современных UI интерфейсов (2024)

* **Blur-эффект**. На фон ставится какое-то изображение и к нему применяется blur. Основная функция изображения — передать цвет и создать иллюзию глубины.
* **Прозрачные элементы**. Некторые элементы делают частично прозрачными или применяют blur-эффект

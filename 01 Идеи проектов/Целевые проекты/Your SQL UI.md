Интерфейс для взаимодействия с SQL базой данных

- Можно подключаться к разным БД (локальным и удаленным)
- Можно писать SQL-запросы
- SQL-запросы можно сохранять (локально или в зашифрованном виде на сервере)
- Доступ к БД должен быть ограничен паролем
- В Пересылке данных между удаленной базой и клиентским приложением необходимо использовать шифрование
- Учитывать ограничения в правах пользователей
- Использовать ORM
- Реализовать интерфейс взаимодействия приложения через терминал

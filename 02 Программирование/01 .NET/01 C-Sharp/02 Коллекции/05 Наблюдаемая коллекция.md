
Тип `System.Collection.ObjectModel.ObservableCollection<T>`

По функциональным возможностям класс похож на класс `List<T>`. Основное отличие состоит в том, что данный класс может извещать внешние объекты об изменениях коллекции. Данный тип коллекции используется в фреймворках WPF, Avalonia, 

| Конструктор                                          | Описание                     |
| ---------------------------------------------------- | ---------------------------- |
| `ObservableCollection<T>()`                          | Создание пустой коллекции    |
| `ObservableCollection<T>(IEnumerable<T> collection)` | Создание на основе коллекции |

| Метод                              | Описание                                                                                                         |
| :--------------------------------- | :--------------------------------------------------------------------------------------------------------------- |
| `Move(int oldindex, int newindex)` | Перемещает элемент со старым индексом на новый индекс, элементы левее нового индекса смещаются к старому индексу |
| `Add(T item)`                      | Добавляет новый элемент                                                                                          |
| `AddRange(IEnumerable<T> items)`   | Добавляет список элементов в коллекцию                                                                           |
| `CopyTo()`                         | Копирует коллекцию в новую коллекцию                                                                             |
| `Contains(T item)`                 | Проверяет, содержит ли коллекция данный элемент                                                                  |
| `Clear()`                          | Очищает коллекцию                                                                                                |
| `IndexOf(T item)`                  | Возвращает индекс данного элемента                                                                               |
| `Insert(T item)`                   | Вставляет элемент по индексу                                                                                     |
| `Remove(T item)`                   | Удаляет элемент по значению                                                                                      |
| `RemoveAt(int index)`              | Удаляет элемент по индексу                                                                                       |


Можно создавать собственные псевдонимы, которые можно использовать по аналогии с макроподстановками в языках программирования (например макроподстановки в C++: `#define`).

В данном примере, задается параметр конфигурации Git на глобальном уровне. Задается псевдоним `ignore-list`, который будет заменяться при использовании на `git ls-files --others -i --exclude-standard`:

```git
git config --global alias.ignore-list "! git ls-files --others -i --exclude-standard"
```

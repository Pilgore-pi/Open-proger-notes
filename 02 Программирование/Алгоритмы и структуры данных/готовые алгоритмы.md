## Пузырьковая сортировка

```cs
void Bubble_Sort(int[] arr)
{
	for (int i = 0; i < arr.Length; i++)
		for (int j = 0; j < arr.Length - 1 - i; j++)
			if (arr[j] > arr[j + 1])
			{
				Swap(ref arr[j], ref arr[j + 1]);
			}
}

void Swap(ref int var1, ref int var2)
{
	int temp = var1;
	var1 = var2;
	var2 = temp;
}
```
  
Сложность – n2. Пузырьковая сортировка, сортировка простых обменов. Принадлежит классу сортировок обменами. Механизм: происходит последовательное сравнение двух ближайших элементов. Если левый элемент больше правого, то они меняются местами. Сравнение происходит снова и снова, пока массив не будет отсортирован по возрастанию.

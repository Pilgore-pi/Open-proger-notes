
Буфер обмена Windows может содержать данные в следующих форматах (WinAPI):


| Идентификатор    | Описание         | Формат                                    |
| ---------------- | ---------------- | ----------------------------------------- |
| `CF_UNICODETEXT` | Текстовые данные | Текст всегда храниться в кодировке UTF-16 |
| `CF_HDROP`       |                  |                                           |
| `CF_BITMAP`      |                  |                                           |

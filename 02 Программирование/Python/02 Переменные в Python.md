## Переменные в Python

### Идентификаторы переменных

- Переменная — это именованное значение в памяти, которому можно присвоить данные и обращаться по имени.
- Имя переменной (идентификатор) должно начинаться с буквы или символа подчеркивания `_`.
- Имя может содержать буквы (любые Unicode), цифры и символы подчеркивания.
- Python чувствителен к регистру: `var`, `Var` и `VAR` — это разные переменные
- Не допускаются имена, совпадающие с ключевыми словами языка (например, `if`, `for`, `while`).
- В Python существует общепринятый стиль именования переменных и функций — snake_case (нижние буквы с подчеркиваниями) для имен переменных

Пример корректных имён:

```python
my_variable = 10
_underscore_var = 20
variable123 = 30
```

Недопустимые имена:

```python
123var = 10  # не может начинаться с цифры
for = 5  # ключевое слово
my-variable = 100  # дефис недопустим
```

### Присвоение значений

- В Python переменные создаются сразу при присвоении значения (нет необходимости объявлять тип).
- Тип переменной определяется автоматически в момент присвоения.
- Переменная может менять тип в процессе выполнения программы (динамическая типизация).

```python
x = 10        # целое число
x = "string"  # теперь строка
```

## Основные типы данных в Python

### Числовые типы

| Тип       | Описание                               |
| --------- | ------------------------------------ |
| int       | Целые числа произвольной точности    |
| float     | Числа с плавающей точкой (двойная точность) |
| complex   | Комплексные числа (с двумя вещественными частями) |

- Целые числа `int` в Python неограничены в размере, зависят только от доступной памяти.
- `float` — числа с плавающей точкой двойной точности (аналог 64-бит double в C#).
- Комплексные числа имеют вид `a + bj`, где `a` и `b` — вещественные числа.

### Логический тип

| Тип    | Описание                   |
| ------ | -------------------------- |
| bool   | Логическое значение `True` или `False` |

### Строки

- Строки в Python — это последовательность символов Unicode.
- Строки могут быть заключены в одинарные `'...'`, двойные `"..."` или тройные кавычки `'''...'''`/`"""..."""` для многострочных строк.
- Строки неизменяемы (immutable).

## Литералы в Python

### Числовые литералы

- Целые числа: `0`, `123`, `-456`, `0b101` (двоичное), `0o77` (восьмеричное), `0x9F` (шестнадцатеричное)
- Числа с плавающей точкой: `3.14`, `-0.001`, `1e9` (экспоненциальная форма)
- Комплексные: `1+2j`, `-3.5j`

```python
a = 10
b = 0b1010  # двоичное 10
c = 3.14
d = 1e-3
e = 3 + 4j
```

### Строковые литералы

```python
s1 = 'hello'
s2 = "world"
s3 = '''Многострочная
строка'''
s4 = """Ещё
одна многострочная строка"""
```

- Экранирование символов: `\n` (новая строка), `\t` (табуляция), `\\` (обратный слеш), и др.

### Булевы литералы

- `True` и `False` — литералы булевых значений.

### Специальный литерал None

- `None` — специальное значение, означающее "отсутствие значения" или "пусто".

## Особенности работы с переменными и литералами

- В Python переменные не требуют явного указания типа.
- Литералы — это конкретные значения, указанные прямо в коде.
- Все переменные — это ссылки на объекты в памяти.
- Тип переменной можно узнать функцией `type()`.

```python
x = 42
print(type(x))  # lassss 'int'>

y = "text"
print(type(y))  # lass 'strtr'>
```

что-то ничего подходящего не могу найти из готовых вариантов. Я умею программировать и не боюсь экспериментов. составь план работ по сборке 10 программируемых автономных светильников Arduino. Светильники должны быть тусклее, чем стандартные лампочки, так как стандартные лампочки используются, обычно, в единственном экземпляре, а я хочу равномерно распределённый свет. Также, я хочу создать (или найти готовое) простейшее приложение по управлению этими светильниками

Для создания 10 программируемых автономных светильников на Arduino с равномерным распределением света и управлением через приложение, следуйте этому плану:

---

### 1. **Выбор компонентов**

- **Arduino Nano или Pro Mini** (компактные версии для экономии места)[4].
- **Светодиодные ленты WS2812B** (адресные, для равномерного свечения и индивидуального управления каждым сегментом).
- **Bluetooth-модули HC-05/06** для беспроводного управления[2][3][5].
- **Литиевые аккумуляторы 18650** (3.7 В, 2000 мАч) + платы защиты и зарядки TP4056.
- **DC-DC преобразователи** (для стабилизации напряжения до 5 В).
- **Корпуса** из пластика или алюминия с креплениями (магниты, двусторонний скотч, клипсы).
- **Резисторы 220 Ом** и **конденсаторы 1000 мкФ** для защиты светодиодов[4].

---

### 2. **Схема подключения**

- Подключите светодиодные ленты к цифровым пинам Arduino через резисторы.
- Bluetooth-модуль соедините с Arduino:
  - **RX (Bluetooth) → TX (Arduino)**.
  - **TX (Bluetooth) → RX (Arduino)**.
  - **VCC → 5 В**, **GND → GND**[2][5].
- Аккумулятор подключите через DC-DC преобразователь к Arduino и светодиодам.

---

### 3. **Программирование Arduino**

1. **Установите библиотеки**:
   - FastLED для управления адресными светодиодами.
   - SoftwareSerial для работы с Bluetooth.
2. **Напишите скетч**:
   - Регулировка яркости через ШИМ (PWM) для снижения интенсивности света[4].
   - Обработка команд от Bluetooth (например, изменение яркости, включение/выключение).

```cpp
#include <FastLED.h>
#include <SoftwareSerial.h>

#define NUM_LEDS 10
SoftwareSerial bluetooth(10, 11); // RX, TX
CRGB leds[NUM_LEDS];

void setup() {
    FastLED.addLeds<WS2812B, 6, GRB>(leds, NUM_LEDS);
    bluetooth.begin(9600);
}

void loop() {
    if (bluetooth.available()) {
        
        char command = bluetooth.read();
        
        if (command == 'L')              // Уменьшить яркость
            FastLED.setBrightness(50);   // Уровень 50 из 255
        
        else if (command == 'H')         // Увеличить яркость
            FastLED.setBrightness(150);
        
        FastLED.show();
    }
}
```

---

### 4. **Создание приложения для управления**

Используйте **MIT App Inventor**[3]:

1. Добавьте кнопки для управления яркостью и переключения режимов.
2. Настройте Bluetooth-соединение с модулем HC-05/06 (пароль: 1234).
3. Пример логики:
   - При нажатии кнопки "Тусклый свет" отправляйте символ **'L'**.
   - Для "Яркий свет" - символ **'H'**.

---

### 5. **Сборка и монтаж**

1. Закрепите компоненты в корпусе, изолируя контакты.
2. Установите аккумулятор и DC-DC преобразователь.
3. Распределите светильники на стенах/потолке, используя магниты или липучки для равномерного освещения[1].

---

### 6. **Оптимизация энергопотребления**

- Добавьте в скетч режим "сна" для Arduino при отсутствии команд.
- Используйте **выключатели** на корпусах для ручного отключения.

---

### 7. **Тестирование**

- Проверьте равномерность света, затемнив помещение.
- Протестируйте работу Bluetooth на расстоянии до 10 метров[5].
- Измерьте время автономной работы (ожидаемый результат: 8–12 часов при яркости 50%).

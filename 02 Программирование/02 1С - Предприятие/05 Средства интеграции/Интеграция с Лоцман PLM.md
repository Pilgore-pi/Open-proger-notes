# Импорт данных в Лоцман

Способы импорта:

- Через подключение к базе данных (через строку подключения)

Алгоритм импорта данных в Лоцман можно описывать с помощью встроенных инструментов и с помощью скриптов VBS

Сам алгоритм импорта храниться в файле формата`.idx`

\* Дизайнер алгоритма импорта

Импорт данных из сторонних источников в систему ЛОЦМАН:PLM производится модулем Лоцман:PLM (далее Модуль импорта).

**Модуль импорта** выполняет алгоритм импорта данных, содержащийся в файле формата
*.ldx, и создает XML-файл. Данные XML-файла импортируются в систему ЛОЦМАН:PLM.
Модуль импорта можно вызвать из модуля **Дизайнер алгоритма импорта** для выполнения текущего алгоритма или запустить как самостоятельное
приложение системы ЛОЦМАН:PLM.
Исполняемый файл модуля Модуль импорта – `WinImport.exe`

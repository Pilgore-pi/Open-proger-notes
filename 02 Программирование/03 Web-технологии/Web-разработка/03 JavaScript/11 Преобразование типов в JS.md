```js
// в строку
String(true);

// в число
'1' / '2' // 0.5
6 - '2' // 4
Number('2'); // 2
Number('dflksdf'); // NaN
Number(true); // 1
Number(null); // 0
Number(undefined); // NaN
+"2" // 2
+true // 1
+"" // 0

// в Boolean
Boolean(0); // false
Boolean(1); // true
Boolean(""); // false
Boolean("creep"); // true (любая непустая строка)
```

При преобразовании строки в число с помощью Number(), все пробелы и управляющие символы убираются (по краям), затем происходит попытка преобразования.

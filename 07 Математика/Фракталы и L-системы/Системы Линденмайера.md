В 1968 году венгерский биолог Аристид Линденмайер для моделирования роста растений и других организмов разработал систему формальных грамматик (L-системы). L-системы — это описание механизма итеративного роста

Структура L-систем:

- Алфавит — множество используемых символов в грамматике
- Аксиома — начальное состояние (начальноя строка), представляющая какое-то множество символов из алфавита
- Правила — определяют какие символы будут заменены на подстроки. Например:

| Алфавит | $\{A, B, C\}$                                                 |
| ------- | ------------------------------------------------------------- |
| Аксиома | $A$                                                           |
| Правила | $A \rightarrow AB$<br>$B \rightarrow CA$<br>$C \rightarrow C$ |

6 итераций:

| A               |
| --------------- |
| AB              |
| ACA             |
| ABCAB           |
| ABCACABCA       |
| ABCACABCABCACAB |
| ...             |

#### Кривая Коха

| Алфавит | {F, +, -}                                                           |
| ------- | ------------------------------------------------------------------- |
| Аксиома | $F$                                                                 |
| Правила | $F \rightarrow F+F-F-F+F$<br>$+ \rightarrow +$<br>$- \rightarrow -$ |

В данном примере можно разделить символы алфавита на переменные (`F`) и константы (`+, -`)

Можно легко отобразить фрактал с помощью черепашей графики. Плюс будет означать поворот на 90 градусов, а минус на -90 градусов. Символ F будет означать черту некоторой длины.

Это простейший пример. Можно добавлять свои символы, менять угол поворота, добовляя скосы. Символ может означать не только отрисовку черты, но и отрисовку многоугольника или более сложной фигуры

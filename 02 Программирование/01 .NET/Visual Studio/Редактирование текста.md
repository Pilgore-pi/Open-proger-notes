## Комментирование

Комментирование выделенного текста: `Ctrl + (K + C)`.
Расскоментированние выделенного текста: `Ctrl + (K + U)`

## Поиск и замена текста

Поиск текста: `Ctrl + F`.
Переход к другому вхождению текущего слова: `Ctrl + Shift + UP/DOWN`
Поиск и замена текста: `Ctrl + H`
При поиске текста можно использовать регулярные выражения

## Строки кода

Удаление строки: `Shift + Del`
Дублирование строки: в любом месте строки: `Ctrl + D`
Копирование и вставка выделенного участка на текущей строке: `Ctrl + D`
Перемещение строки вверх: `Alt + UP`
Перемещение строки вниз: `Alt + DOWN`

## Завершение блока

При написании блока if (...), while(...) и др. Можно нажать `Shift + Enter`. Таким образом создадутся необходимые скобки и курсор сразу перейдет в позицию между скобками.

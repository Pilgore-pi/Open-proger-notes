Условия:

```js
// if выполняется если expression == true
if(expression) Action();
if(expression){ Action(); }
else { }

// Тернарный оператор
let result = условие ? значение1 : значение2;

// Оператор switch-case
switch(x){
	case 'value1':
		...
	break;
	...
	[default:]
}
```

Циклы:

```js
// Цикл while
while(condition) {...}

// do while
do {...} while(condition);

// Цикл for
for(let i = 0; i < 13; i++) {...}
// оператор continue как в C#

// Перебор объекта
for(let property in obj) {...}
// тело цикла выполнится для каждого свойства объекта

// Перебор коллекции (числовых свойств объекта)
for(let item of collection) {...}
```

>Цикл `for in` может быть в 10-100 раз медленнее чем цикл `for of` для коллекций.

>В цикле `for of` перебираемые элементы доступны только для чтения

>Для массивов допустимы отрицательные индексы, любые неинициализированные элементы будут `undefined`

> **JavaScript** — это язык программирования для Web с динамической слабой типизацией. Поддерживает ООП и функциональный стиль. Автоматическое управление памятью.

В JS используется ***прототипное программирование***.

>В браузере можно открыть средство просмотра кода страницы (F12 или Ctrl + U), которое позволяет видеть сообщения об ошибках и писать код JS в консоль.

## Использование кода в HTML

#### Простейший пример **встроенного кода**:
```html
<!DOCTYPE HTML>
<html>
<body>
  <p>Перед скриптом...</p>
  <script>
    alert( 'Привет, мир!' );
  </script>
  <p>...После скрипта.</p>
</body>
</html>
```

~У тега script есть атрибут "language", который по умолчанию равен "JavaScript".

Тег `<script>` содержит JavaScript-код, который автоматически выполнится, когда браузер его обработает.

#### Пример внешнего кода (в отдельном файле .js):

Ссылка на локальный файл:

```html
<script src="/path/to/script.js"></script>
```

Ссылка на URL:

```html
<script src="https://...lodash.js"></script>
```

Если атрибут src установлен, то тело тега будет игнорироваться.

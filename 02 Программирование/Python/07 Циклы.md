
## Оператор **while**

```python
while condition:
    # тело цикла
```

Условие проверяется перед каждой итерацией. Цикл выполняется, пока условие истинно

## Оператор **for**

```python
for variable in iterable:
    # тело цикла
```

Цикл перебирает элементы итерируемого объекта (`list`, `tuple`, `string`, `range` и т.д.)

### Особенности:

- Для создания диапазона чисел обычно используется функция `range()`:

```python
for i in range(9):
    print(i)
```

- Можно задать старт, стоп и шаг аналогично:

```python
for i in range(0, 10, 2):
    print(i)
```

### Бесконечный цикл:

```python
while True:
    AcceptClients()
```

Для выхода из цикла используется оператор **break**

## Переменные внутри циклов

Переменная, объявленная в теле цикла, доступна и после цикла: область видимости — функция или глобальная область, а не только цикл

## Итерация по коллекциям и строкам

В Python оператор `for` используется для итерации по любой коллекции (списки, строки, множества, словари и т.д.).

```python
fibNumbers = [0, 1, 1, 2, 3, 5, 8, 13]

for element in fibNumbers:
    print(element, end=" ")
# Output: 0 1 1 2 3 5 8 13
```

#### Итерация по многомерным спискам

Если список вложенный, например, список списков — итерация проходит по "верхнему слою":

```python
matrix = [[1, 2], [3, 4]]

for row in matrix:
    print(row)
# Output:
# [1, 2]
# [3, 4]
```

Если нужно пройти по всем элементам, требуется вложенный цикл

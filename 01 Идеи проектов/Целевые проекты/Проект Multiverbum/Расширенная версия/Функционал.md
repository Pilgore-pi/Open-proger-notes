> Никаких пользователей и паролей, никакой фонетики и речевого ввода, никакого перевода

### Типы пользователей  
  
Разграничение прав доступа на 3 роли: администратор, учитель и ученик.  

- **Администратор**: имеет полный доступ ко всем материалам и может изменять уровень доступа других пользователей.
- **Учитель**: может создавать теоретические материалы для изучения и практические материалы для тренировки и тестирования. Может изменять и удалять свои материалы. Может повысить любого ученика до статуса учителя.
- **Ученик**: может только использовать теоретические и практические материалы.

Все пользователи могут, как минимум, использовать теоретические и практические материалы.

### Основные функции  

1. Создание, изменение, использование и удаление обучающих материалов (тесты, теоретические статьи, упражнения). Все материалы могут быть оценены по качеству любыми пользователями.
2. Возможность определения уровня владения английским языком
3. **Фонетический раздел** (произношение и прослушивание слов)
4. **Чат**. Возможность общения между пользователями приложения
5. **Составление личного плана**. Любой пользователь может составить план обучения для самого себя, который может включать теоретические материалы, тесты и упражнения. Пользователь имеет возможность отслеживать прогресс личного плана по мере изучения материалов.
6. **Общение с чат-ботом** с искусственным интеллектом (практика устного и письменного английского). Чат-бот может произносить свои сообщения вслух по желанию пользователя.
7. **Синтез речи**. Во всех разделах приложения, где пользователь может вводить текст, он также может использовать устную речь (синтез речи)
8. ПЕРЕВОД. Допускается возможность перевода сразу на несколько языков и вывод таблицы с результатами. Добавить в таком случае кнопку для копировании таблицы в синтаксисе Markdown

- Информация обо всех пользователях и обучающих материалах будет храниться в базе данных.

- Практические материалы представляют собой набор разнотипных заданий (какого именно типа задание определяет учитель) за которые начисляются баллы. Все практические материалы разбиты по категориям и подкатегориям, которые также могут иметь подразделы.

Существует 2 типа практических материалов:

1. Тренировочные задания — разнотипные задания с оценкой для проработки данной темы, доступны подсказки и раскрытие ответа.
2. Тесты — оценка знаний пользователя по данной теме, подсказки недоступны
  
### Уникальность проекта  
  
Мой проект отличается от других аналогов, благодаря возможности общения между пользователями приложения и непосредственного влияния на качество обучающих материалов. Кроме того при отсутствии материалов на какую-либо тему пользователи могут изучить эту тему и поделиться своими знаниями с другими пользователями, создав соответствующие материалы.

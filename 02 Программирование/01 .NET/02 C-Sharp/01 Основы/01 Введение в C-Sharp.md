> **Язык программирования C#** — это объектно-ориентированный язык программирования общего назначения, разработанный компанией Microsoft в 2002 году.

Автор языка — Андерс Хейлсберг, создатель языка **Turbo Pascal**, главный архитектор **C#** и создатель **TypeScript**

Язык C# является:

* Компилируемым
* Императивным
* Высокоуровневым
* Мультипарадигменным:
    * Объектно-ориентированным
    * Функциональным
    * Событийно-ориентированным
* Статически типизированным (поддерживается динамическая типизация)
* Сильно типизированным
* Явно типизированным
* Языком общего назначения

На [GitHub](https://github.com/dotnet/csharplang) есть открытый исходный код компилятора языка C#

## Платформа .NET

> Язык C\# подчиняется универсальной платформе разработки приложений .NET

Платформа .NET определяет единую систему типов и берет на себя взаимодействие с ОС, предоставляя удобный программный интерфейс для разработки приложений.

Существует несколько *нативных* языков программирования, которые были изначально разработаны на основе .NET:

- C\#
- F\#
- Visual Basic .NET (ранее язык Basic не входил в стандарт .NET)
- PowerShell

Другие же языки напрямую не поддерживают стандарт .NET, но имеют версии, которые задействуют платформу .NET и выполняют программы на ней


| Язык   | Реализация с .NET                                    |
| ------ | ---------------------------------------------------- |
| Python | IronPython                                           |
| C++    | [C++\CLI](https://en.wikipedia.org/wiki/C%2B%2B/CLI) |
| Ruby   | IronRuby                                             |
| Java   | J\# (Устрел с 2015 года)                             |
| Pascal | PascalABC.NET                                        |
| Delphi | Delphi.NET                                           |

> **Язык программирования C** — это процедурный язык программирования общего назначения

> **Язык программирования C++** — это объектно-ориентированный язык программирования общего назначения, разработанный Бьярном Страуструпом в 1983 году как расширение языка C.

Автор языка — Бьярн Страуструп (Bjarne Stroustrup), датский программист, который продолжает активно участвовать в развитии и стандартизации языка по сей день.

Язык C++ является:

* Компилируемым
* Императивным
* Низкоуровневым (с возможностью высокоуровневого программирования)
* Мультипарадигменным:
    * Объектно-ориентированным
    * Процедурным
    * Функциональным
    * Обобщённым (Generic Programming)
    * Метапрограммированием
* Статически типизированным
* Сильно типизированным
* Явно типизированным
* Языком общего назначения

На [GitHub](https://github.com/cplusplus) есть открытые ресурсы и стандарты языка C++

## История и стандартизация

C++ развивается через серию стандартов, каждый из которых добавляет новые возможности:

| Стандарт | Год | Основные особенности |
| -------- | --- | -------------------- |
| C++98    | 1998 | Первый стандарт ISO |
| C++03    | 2003 | Исправления и уточнения |
| C++11    | 2011 | Лямбда-функции, auto, умные указатели |
| C++14    | 2014 | Улучшения C++11 |
| C++17    | 2017 | Структурированные привязки, std::optional |
| C++20    | 2020 | Концепции, корутины, модули |
| C++23    | 2023 | Дальнейшие улучшения и оптимизации |

## Особенности C++

### Управление памятью

C++ предоставляет прямой контроль над управлением памятью:

- Ручное выделение и освобождение памяти (new/delete)
- Указатели и ссылки
- Умные указатели (smart pointers) в современном C++
- Стек и динамическая память

### Производительность

- Минимальные накладные расходы абстракций ("zero-cost abstractions")
- Возможность написания кода близкого к ассемблеру
- Отсутствие встроенной сборки мусора (контролируется программистом)
- Оптимизация на уровне компилятора

### Стандартная библиотека (STL)

C++ включает мощную Стандартную библиотеку шаблонов:

- Контейнеры (vector, list, map, set)
- Алгоритмы (sort, find, transform)
- Итераторы
- Функциональные объекты

## Область применения

- Системное программирование (ОС, драйверы, встроенные системы)
- Высокопроизводительные приложения
- Разработка игр (движки, графика)
- Финансовые системы и высокочастотный трейдинг
- Обработка больших данных и научные вычисления
- Компиляторы и интерпретаторы
- Базы данных
- Графические приложения и 3D визуализация
- IoT и встроенные системы
- Приложения реального времени

## Сравнение C++ с C#

| Аспект | C++ | C# |
| ------ | --- | --- |
| Платформа | Кроссплатформенный (компиляция) | .NET Framework / .NET Core |
| Управление памятью | Ручное | Автоматическое (GC) |
| Производительность | Очень высокая | Высокая |
| Кривая обучения | Крутая | Более пологая |
| Скорость разработки | Медленнее | Быстрее |
| Типизация | Статическая, сильная | Статическая, сильная |
| Парадигмы | ООП, процедурная, функциональная, метапрограммирование | ООП, функциональная, событийная |

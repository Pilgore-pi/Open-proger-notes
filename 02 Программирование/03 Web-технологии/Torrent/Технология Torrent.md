## Торрент файл

Содержит метаданные, включающие ПО, на котором файл был создан, автора и другую информацию.

В торрент-файле определено несколько типов, которые обозначаются открывающими и закрывающими флагами:
* Строка
* Число
* Массив
* Ассоциативный массив

## Технология Torrent

В сети каждый клиент, получающий или уже получивший все файлы также является сервером, который раздает файлы.

Файлы деляться на сегменты, при этом один сегмент может содержать как конец одного файла, так и начало следующего. Поэтому если пользователь хочет скачать лишь указанный файл, то ему может потребоваться также скачать кусочек другого файла.

* **Пир** (peer) — компьютер, который находится в процессе скачивания и раздачи одновременно
* **Сид** — компьютер, который завершил скачиванием и занимается только раздачей
* **Лич** (leech) — компьютер, который желает только скачать файл
* **Трекер** — специальный сервер, который передает адреса пиров и сидов клиенту, для получения сегментов файлов.

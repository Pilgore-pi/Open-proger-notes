## Оператор if

Версия Scala 2.0

```scala
if (cond) body

if(cond)
	body

if(cond){
	body
}
else {
	body
}
```

Версия Scala 3.0

```scala
if cond then
	body
	body
[else]
```

```scala
val minValue = if a < b then a else b
```
## Оператор while

Версия 2.0

```scala
while(condition)
	body

while(condition){
	body
}
```

Версия 3.0

```scala
while condition do
	body
//аналогично
```

- Нет цикла `do-while`

## Оператор for

Версия 2.0

```scala
for(variable <- collection){
	body
}	

// Встроенные примитивные коллекции
1 to 10    [1; 10]
'a' to 'z'
1 until 10 [1; 10)
1 to 10 by 5
```

Версия 3.0

```scala
for variable <- collection do
	body
	body
```

```scala
for
	i <- 1 to 10 by 2
	j <- i to 10
	do {
		body
	}
```

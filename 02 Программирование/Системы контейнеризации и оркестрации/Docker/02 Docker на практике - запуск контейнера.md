
Запустим контейнер готового образа с DockerHub. Для этого нужно выполнить следюущие шаги:

1. Скачать приложение Docker Desktop с официального сайта
2. Запустить приложение
3. В строке поиска нужно ввести название тестового образа "`welcome-to-docker`"
4. Найденный образ можно скачать ("`Pull`") или скачать и запустить ("`Run`")
5. При запуске образа отобразится окно подтверждения, где можно будет задать настройки для контейнера

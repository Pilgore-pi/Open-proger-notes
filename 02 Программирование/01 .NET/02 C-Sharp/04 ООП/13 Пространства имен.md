**Пространства имен (namespace) в C#** — это логические контейнеры, которые позволяют группировать связанные классы, интерфейсы, структуры, перечисления и другие типы, чтобы организовать код и избежать конфликтов имен. Они обеспечивают уникальность идентификаторов в пределах проекта и помогают структурировать большие проекты, делая код более читаемым и поддерживаемым

Объявление: используется ключевое слово namespace, за которым следует имя пространства имен. Внутри него размещаются классы и другие типы.

```csharp
namespace MyNamespace
{
    class MyClass
    {
        public void MyMethod()
        {
            Console.WriteLine("Пример пространства имен");
        }
    }
}
```

Можно также объявить пространство имен на уровне файла:

```csharp
namespace MyNamespace;

class MyClass
{
    public void MyMethod()
    {
        Console.WriteLine("Пример пространства имен");
    }
}
```

**Доступ к членам:** чтобы использовать классы из пространства имен, можно писать полное имя с использованием точки: `MyNamespace.MyClass`. Для упрощения часто используется директива `using`, которая позволяет не писать полное имя

> **Вложенные пространства имен:** пространства имен могут содержать другие пространства имен, что позволяет создавать иерархическую структуру

**Глобальные пространства имен:** начиная с C# 10, можно использовать `global using` для автоматического подключения часто используемых пространств имен во всем проекте

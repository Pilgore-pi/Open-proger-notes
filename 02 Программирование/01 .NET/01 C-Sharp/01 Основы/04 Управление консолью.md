Управление консолью происходит через класс `Console`, предоставляющий необходимые методы для ввода-вывода в консоль

| Метод          | Возвращает                | Описание                                                                                      |
| -------------- | ------------------------- | --------------------------------------------------------------------------------------------- |
| `Write()`      | `void`                    | выводит текстовое представление объекта в консоль                                             |
| `WriteLine()`  | `void`                    | Аналогичен `Write()`, но в конец, к выводу, добавляет перевод на новую строку                 |
| `Read()`       | `int`                     | Считывает первый символ из введенной строки и возвращает код этого символа в кодировке UTF-16 |
| `ReadLine()`   | `string?`                 | Считывает введенный пользователем текст как строку и возвращает ее                            |
| `ReadKey()`    | `(struct) ConsoleKeyInfo` | Считывает введенную клавишу и возвращает информацию о ней в виде структуры `ConsoleKeyInfo`   |
| `Clear()`      | `void`                    | Очищает буфер консоли                                                                         |
| `Beep()`       | ``                        |                                                                                               |
| `ResetColor()` | ``                        |                                                                                               |

Если метод WriteLine() принимает несколько параметров, то первым должен быть string или `char[]`

**Свойства:**

1) ForegroundColor – цвет текста
2) BackgroundColor – цвет фона консоли (принимает зн-е ConsoleColor)
3) InputEncoding – кодировка консоли, используемая при чтении входных данных
4) Title – название консоли
5) WindowHeight
6) WindowWidth

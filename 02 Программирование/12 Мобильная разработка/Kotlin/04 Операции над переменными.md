
| Операция | Описание  | Количество операндов |
| -------- | --------- | -------------------- |
| `+` | Сложение | Бинарная |
| `-` | Вычитание | Бинарная |
| `*` | Умножение | Бинарная |
| `/` | Деление  | Бинарная |
| `%` | Остаток от деления | Бинарная |
| `c++` | Постфиксный инкремент. Возвращает `с`, затем прибавляет `1` к `c`  | Унарная |
| `++c` | Префиксный инкремент. Возвращает `с+1` и прибавляет `1` к `c` | Унарная |
| `c--` | Постфиксный декремент. Возвращает `с`  | Унарная |
| `--c` | Префиксный декремент. Возвращает `с-1` | Унарная |
| `&` | Логическое **_И._** Для чисел выполняет операцию для каждого бита. | Бинарная |
| `&&`  | Логическое **_И_** только для булевых значений. Не выполняет проверку на истинность, если левый операнд уже имеет значение `false`. | Бинарная |
| \|  | Логическое **_ИЛИ._** Для чисел выполняет операцию для каждого бита. | Бинарная |
| \|\|  | Логическое **_ИЛИ_** только для булевых значений. Не выполняет проверку на истинность, если левый операнд уже равен `true`. | Бинарная |
| `!` | Логическое **_НЕ_** | Унарная |
| `=` | Присваивание значения | Бинарная |
| `==`  | Логическая равенство  | Бинарная |
| `^` | Исключающее **_ИЛИ._** Для чисел выполняет операцию для каждого бита. | Бинарная |
| `~` | Инверсия каждого бита числа  | Унарная |
| `shr`  | (`>>`, shift right) Сдвигает 1-й операнд на (2-й операнд) двоичных разрядов вправо  | Бинарная |
| `shl`  | (`<<`, shift left) Сдигает 1-й операнд на (2-й операнд) двоичных разрядов влево  | Бинарная |
| `ushr`  | (`<<`, unsigned shift right) Аналог `shr` для беззнаковых чисел  | Бинарная |
| `and`  | Побитовое И | Бинарная |
| `or`  | Побитовое ИЛИ | Бинарная |
| `xor`  | Побитовое исключающее ИЛИ | Бинарная |
| `inv()`  | Побитовое отрицание | Бинарная |

Сокращенные операции

| Сокращенная операция | Эквивалент |
| -------- | --------- |
| `A += B` | `A = A + B` |
| `A -= B` | `A = A - B` |
| `A *= B` | `A = A * B` |
| `A /= B` | `A = A / B` |
| `A %= B` | `A = A % B` |

## Операции преобразования

Любое явное преобразование одного типа к другому осуществляется через специальные встроенные функции

| Функция | Описание |
| ------- | -------- |
| `Число(строка)`  | Строка в число |
| `Формат(объект, форматнаяСтрока)` | Преобразует простое значение в форматированный текст |
| `Дата(строка)`   | Преобразует строку в дату и время |
| `Строка(объект)` | Преобразует значение в строку. Для примитивных типов применяет формат по умолчанию (функция `Формат()`) |
| `Булево(строка)` | Текст в булево |

```bsl
// Преобразования к числу


// Преобразования к строке
//     Явные:

Строка(ложь);
Строка(истина);
Строка(0);
Строка(1);
Строка(1.123456789);
Строка('2025-12-01');

справочникСсылка = получитьСсылкуНаСправочник();
Строка(справочникСсылка);

//    Неявные:
//    (К строке)
""   + 12;                // Стандартный формат числа 12
""   + 1.2;               // "1,2"; стандартный формат числа 1.2
""   + ложь;              // Стандартный формат булево
""   + справочникСсылка;  // Представление ссылки
""   + Дата(2025, 12, 1); // Стандартный формат даты (дд.ММ.ГГГГ чч:мм:сс)

//    (К числу)
12   + "";                // Исключение

//    (К объектному типу)
справочникСсылка + "";    // Исключение

```

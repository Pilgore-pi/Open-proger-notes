`Nothing` — наследник всех вышестоящих типов, не имеет ни одного значения

Все числовые типы являются знаковыми и представляют типы значения:

- `Byte` (1 байт): `-128` до `127`

- `Short` (2 байта): `-32768` до `32767`

- `Int` (4 байта): `-2^31` до `2^31-1`

- `Long` (8 байт): `-2^63` до `2^63-1`

- `Float` (4 байта): числовой тип с плавающей точкой одинарной точности

- `Double` (8 байт): числовой тип с плавающей точкой двойной точности

- `BigInt` и `BigDecimal` — произвольная точность (ограничена только памятью)

Числовые литералы:

```scala
10           // десятичное целое
-78          // отрицательное число
0x54c        // шестнадцатеричное число
2_100_000    // чтение упрощено подчеркиваниями
85l          // литерал типа Long ('L' или 'l' в конце)
12.3f        // Float литерал
BigInt(25)   // конструктор для произвольно больших целых
```

Логические значения:

```scala
true
false
```

`Unit` — полноценный тип данных, имеющий **единственное** значение `()`. Аналог `void` в других языках, но `Unit` — реальный тип.

## Операции

- Арифметические: `+`, `-`, `*`, `/`, `%`

- Побитовые операции для целочисленных типов:
  - OR: `|`
  - AND: `&`
  - NOT: `~`
  - XOR: `^`

- Логические операции:

  - ИЛИ: `||`
  - И: `&&`
  - Равенство: `==`
  - Неравенство: `!=`

> Оператор `==` в Scala сравнивает содержимое, а не ссылки (сравнение значений)

Все текстовые типы (например, `String`) используют кодировку UTF-16.

  

Пример строк:

  

```

  

// Строки в кавычках

val s1 = "Hello, Scala!"

val s2 = """Многострочная

строка, где все символы

воспринимаются буквально,

включая переносы строк"""

  

```

  

Интерполяция строк — удобный способ вставить значения переменных:

  

```

  

val x = 10

println(s"x = \$x")   // вывод: x = 10

println(s"x + 1 = \${x + 1}")  // вложенное выражение

  

```

  

Scala сравнивает строки по содержимому:

  

```

  

"abc" == ("a" + "bc")  // true

  

```

Коллекции в Python подразделяются на следующие категории:

https://habr.com/ru/articles/319164/

1. Последовательности (sequences)
    - Изменяемые (mutable): list (связный список)
    - Неизменяемые (immutable): string (строка), tuple (кортеж)
2. Множества (sets)
    - Изменяемые: set (множество)
    - Неизменяемые: frozenset ("замороженное" множество)
3. Отображения (Mappings)
    - Словарь — dict

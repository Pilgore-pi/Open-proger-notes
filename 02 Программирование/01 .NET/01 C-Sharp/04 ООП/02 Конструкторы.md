
> Каждый раз, когда создается класс или структура, вызывается конструктор.

> **Конструктор** — это специальный метод класса, который неявно вызывается при создании объекта через ключевое слово `new`.

> Конструктор не может иметь возвращающего значения

## Стандартный конструктор

> **Стандартный конструктор**. Если программист не определил конструктор в классе, то компилятором C# (не средой .Net) будет определен конструктор без параметров неявным образом

## Специальный конструктор

> Специальный конструктор — это конструктор, определенный программистом

> Как только программист определяет специальный коструктор, стандартный конструктор удаляется из класса

Если класс/структура содержит только один приватный конструктор, то объект "снаружи" создать будет невозможно

## Деструкторы

Деструкторы, в подавляющем большинстве случаев, не используются, так как объекты удаляются автоматически сборщиком мусора

> Деструкторы вызываются перед тем, как объект будет удален из оперативной памяти

## Примеры кода

С версии C# 9 появилась возможность вызова конструктора, с исключением из него названия типа:

```cs
Person tom = new();          // аналогично new Person();
Person bob = new("Bob");     // аналогично new Person("Bob");
Person sam = new("Sam", 25); // аналогично new Person("Sam", 25);
```

В .NET все переменные явлются объектами, поэтому любую переменную можно создать через ключевое слово `new`

```cs
int  i = new int(1);
char c = new char('p');
```

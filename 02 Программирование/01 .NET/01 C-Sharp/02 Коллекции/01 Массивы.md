> **Массив** — это структура данных фиксированной длины, предназначенная для хранения набора элементов одного типа в непрерывной области памяти. Каждый элемент можно получить по индексу, отсчёт начинается с нуля.

- Все массивы являются ссылочными типами

Массивы бывают нескольких видов:

- Одномерные
- 

```cs
int[] numbers = new int[5]; // массив из 5 элементов типа int
string[] names = { "Anna", "Bob", "Tom" };
var names = new string[] { "Anna", "Bob", "Tom" };
```

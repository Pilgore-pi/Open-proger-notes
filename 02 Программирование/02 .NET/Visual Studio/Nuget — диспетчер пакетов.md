Visual Studio позволяет удобно настраивать пакеты для проекта. Во вкладке "Вид", "Другие окна" есть пункт `Консоль диспетчера пакетов`. Эта консоль представляет собой надстройку над PowerShell и позволяет выполнять команды, связанные с настройкой пакетов.

Обновление всех пакетов до последней версии в рамках текущего проекта:

```
Update-Package
```

### Решение проблемы несовместимости пакетов

Существует несколько подходов к решению проблемы, когда 2 пакета несовместимы:

1. **Самый простой** — отказ от использования несовметимого пакета
2. **Выполнение команды** `Update-Package` для установки последней версии всех пакетов
3. **Установка совметсимой версии** пакета. Иногда можно найти другую версию, которая будет совместима со вторым пакетом
4. **Разделение проектов**. Можно вынести пакетные зависимости в отдельный проект и подключать его в целевой проект, либо напрямую, либо как библиотеку.

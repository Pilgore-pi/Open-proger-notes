
> **Форк (fork)** — это точная копия существующего репозитория, созданная в аккаунте на одном из хостингов Git (GitHub, GitLab). По сути, форк — это новый репозиторий, который содержит весь исходный код и историю изменений оригинального репозитория (upstream), но находится в вашем пространстве имён и полностью под вашим контролем
